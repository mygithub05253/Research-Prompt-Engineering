# GIC 산업 Top Pick 프롬프트 v7.0 — Claude 최적화

> **목적**: 산업 내 후보 비교 → 100점 스코어링 → Top Pick 선정
> **핵심 원칙**: 출처 + CoT + AD-FCoT + Evidence Card + 균형 비교

---

## 모델 메모 (Claude 전용)
- Artifacts로 비교표/레이더차트(SVG) 생성
- AD-FCoT: 과거 유사 산업/기업 사례 참조
- 후보 기업 간 공정한 양면 비교 (편향 방지)

---

## 분석 순서 (Think step by step)

### 1. 산업 분석
- 시장 규모/CAGR + 사이클 위치
- AD-FCoT: "과거 [유사 산업]이 [유사 단계]에서..." → 참조

### 2. 후보 데이터 수집
각 기업: 재무 + 경쟁력 + 뉴스 (모든 수치에 출처)

### 3. 비교 분석
- 정량 비교표 (Artifact)
- 정성 비교표 (경쟁우위/성장성/리스크/모멘텀)

### 4. 100점 스코어링 (CoT: 각 점수의 이유 명시)

| 평가 항목 | 배점 | 기업A | 기업B | 기업C |
|-----------|------|-------|-------|-------|
| 밸류에이션 매력도 | 25 | | | |
| 이익 성장성 | 25 | | | |
| 수익성/효율성 | 15 | | | |
| 경쟁 우위 | 15 | | | |
| 모멘텀/촉매 | 10 | | | |
| 재무 안정성 | 10 | | | |
| **합계** | **100** | | | |

각 점수에 Evidence Card + AD-FCoT 유사 사례

### 5. Top Pick 선정
- 선정 기업 + 근거 3가지
- 간이 밸류에이션 (PER 기반 목표주가)
- 핵심 리스크 + 대응
