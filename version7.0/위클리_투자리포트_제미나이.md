# GIC 위클리 투자리포트 프롬프트 v7.0 — Gemini 최적화

> **목적**: 이번 주 핵심 변화를 중심으로 간결한 투자 업데이트 생성
> **출력물**: ① 핵심 변화 요약 ② 간이 밸류에이션 업데이트 ③ PPTX 3장 분량 요약
> **핵심 원칙**: 팩트 기반 + 출처 필수 + Chain-of-Thought

---

## 모델 메모 (Gemini 전용)
- Google 검색 Grounding으로 이번 주 실시간 데이터 수집
- 검색 쿼리 예시를 각 단계에 명시
- 간결한 업데이트 중심 (전체 리서치가 아닌 주간 변화분)

## 0단계 — 초기 설정
| 항목 | 예시 | 필수 |
|------|------|------|
| 기업명 + 종목코드 | 삼성전기(009150) | 필수 |
| 기준주차 | 2026년 5월 1주차 | 필수 |
| 독자 수준 | 초급/중급/전문가 | 필수 |

---

## 1단계 — 이번 주 데이터 수집 (Google 검색)

**검색 순서 (Think step by step)**:
1. "[기업명] 주가 이번주" → 주가 변동 (시가/종가/등락률)
2. "[기업명] 뉴스 이번주" → 핵심 뉴스 3건
3. "[기업명] 공시 이번주" → 주요 공시
4. "[기업명] 증권사 리포트 최근" → 목표주가 변경
5. "[기업명] 외국인 기관 수급" → 수급 동향

---

## 2단계 — 핵심 변화 Top 3 (CoT)

각 변화별:
```
[변화 #N]
팩트 (What): 무슨 일이 있었는가 — 출처: OOO
영향 (So What): 기업 가치에 어떤 영향 — (사고 과정 명시)
확신도: 상/중/하
모니터링: 다음 주 확인 포인트
```

## 4단계 — 리스크 체크

- 기존 리스크 현황

## 5단계 — 결론 (PPTX 3장 분량)

**Slide 1 — 핵심 요약**
- 한줄 요약
- 투자의견: BUY/HOLD/SELL (유지/변경)
- 이번 주 키워드 3개

**Slide 2 — 핵심 변화 상세**
- Top 3 변화 + 영향도

**Slide 3 — 밸류에이션 + 다음 주 관전**
- 간이 밸류에이션 표
- 다음 주 확인 포인트 3가지
