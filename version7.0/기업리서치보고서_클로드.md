# GIC 기업 리서치보고서 프롬프트 v7.0 — Claude 최적화

> **목적**: 증권사 수준 기업분석 리포트를 Chain-of-Thought + Analogy-Driven Financial CoT(AD-FCoT)로 생성
> **출력물**: ① 5개년 재무모델(표) ② 서술 분석 + Artifacts ③ Assumption Evidence Cards
> **핵심 원칙**: 모든 숫자에 출처, 모든 판단에 근거, 유사 금융사례 참조 추론

---

## 공통 규칙

1. **언어**: 반드시 한국어. 영어 전문용어 한글(영문) 병기
2. **출처 필수**: `(출처: OOO, YYYY.MM)` 형식
3. **Chain-of-Thought**: 매 분석 단계에서 사고 과정을 명시적으로 서술
4. **AD-FCoT (유사 사례 추론)**: 과거 유사한 기업/산업 사례를 참조하여 추론 근거 강화
5. **Assumption Evidence Card**: 주요 가정마다 근거 3건 + 출처 + 신뢰도
6. **균형 서술**: 긍정/부정 양면을 균형 있게 서술
7. **대화형 진행**: 단계별 확인 → 피드백 → 다음 단계

## 1단계 — 데이터 수집

### 수집 항목 (Think step by step)

1. **재무제표** (3개년): IS/BS/CF + 사업부별 매출
2. **시장 데이터**: 주가, 시총, PER/PBR/EV_EBITDA, 52주 고저
3. **사업부 구조**: 각 사업부 매출 비중/성장 추이
4. **피어 그룹**: 3~5개 경쟁사 재무 비교
5. **산업 동향**: 시장 규모, CAGR, 핵심 트렌드
6. **뉴스/공시**: 최근 3개월 핵심 이슈 5건
7. **컨센서스**: 증권사 목표주가 (3개 이상)

> 모든 숫자에 출처 필수. 미확인 수치는 "확인 필요" 표기

---

## 2단계 — 산업 분석 (CoT)

**사고 과정**:
1. 먼저 산업의 글로벌/국내 시장 규모와 성장률을 파악
2. Porter's 5 Forces로 산업 구조 분석
3. 산업 사이클 위치 판단

**AD-FCoT 적용**: "과거 [유사산업]이 [성장/성숙]기에 진입했을 때의 패턴을 참조하면..."

### 출력
- 시장 규모/CAGR 표
- 5 Forces (강도 + 근거)
- 핵심 트렌드 3가지

---

## 3단계 — 기업 심층 분석

### 사고 과정 (CoT)
1. 사업 구조를 파악하고 각 사업부 성장 궤적 분석
2. 경쟁 우위(Moat) 요소 식별 및 지속 가능성 평가
3. 재무 지표를 피어 대비 분석하여 상대적 위치 판단
4. 과거 유사 사례(AD-FCoT)로 투자 논리 검증

### 분석 항목
A. **사업 구조**: 사업부별 매출/이익/성장 (3개년 추이 표 — Artifact)
B. **경쟁 우위**: 5가지 Moat 요소 평가
C. **재무 분석**: 수익성/성장성/안정성/효율성 (피어 비교표 — Artifact)
D. **투자포인트** (3~5개): 각각 Evidence Card + AD-FCoT 유사 사례
E. **리스크** (3~5개): 발생 확률 × 영향도 매트릭스
F. **반대 검증**: 결론을 뒤집는 조건

## 4단계 — 5개년 Forward 밸류에이션 (핵심)

### 4-1. 사업부별 매출 추정 + Evidence Card + AD-FCoT

각 사업부:
```
[Evidence Card + AD-FCoT]
가정: 사업부A 매출 YoY +X%
근거 ①: 산업 CAGR — 출처: OOO
근거 ②: 신규 수주 — 출처: DART
근거 ③: 시장 점유율 — 출처: 증권사
유사 사례: "[과거 기업]이 [유사 상황]에서 [X% 성장]했던 사례" — 출처: OOO
신뢰도: 중
```

### 4-2. 3-Statement 연결 (Artifact로 표 생성)
FY-2 ~ FY5 전체 IS/BS/CF 연결 + Balance Check

### 4-3. Blended CAGR
`50% × 1Y Growth + 30% × 3Y CAGR + 20% × 5Y CAGR`

### 4-4. 4종 밸류에이션
PER / PBR / EV-EBITDA / DCF (WACC 민감도)

### 4-5. Bear/Base/Bull × FY1~FY5 시나리오표
각 시나리오: 핵심 가정 1줄 + 목표주가 + 상승여력

### 4-6. 투자의견
≥+15% → BUY / -10%~+15% → HOLD / <-10% → SELL
