🆕 v11.0 신규 원칙 (W · X · P · M)
W — Web Search (웹검색 의무화)
챗봇 내장 웹검색 활성 + 모든 시장 가격·시총·뉴스에 URL 첨부 의무. v9.0의 "권장"에서 v11.0의 "의무"로 격상.
본 리포트 적용: 현재 주가·피어 멀티플·최근 뉴스 20건 URL 첨부.
X — eXport (CSV 동시 출력)
재무·밸류·피어 표를 csv 코드블록으로 동시 출력. Code Interpreter 사용 시 .xlsx 자동 생성.
본 리포트 적용: csv 7종 (재무 IS/BS/CF, 피어, 시나리오, 민감도, 6-Lens) + Excel 통합 1종.
P — Page-safe HTML (페이지 잘림 방지)
기업리서치·위클리는 A4 세로 (210×297mm), 산업 Top Pick은 A4 가로 (1280×720). 모두 CSS page-break-after: always + break-inside: avoid 적용.
본 리포트 적용: 9페이지 모두 1슬라이드 = 1페이지 보장. 브라우저 인쇄 → PDF 변환 완벽.
M — Meta Prompt (본 리포트)
사용한 프롬프트 기법을 별도 HTML(2~3p)로 설명. 학회 발표 시 "이 리포트는 어떤 AI 프롬프트로 만들어졌는가" 백업 자료.
본 리포트 적용: 본 문서가 그 메타 리포트.

6-Lens 가중치 프레임워크 (Hardik Framework)

본 리포트의 최종 투자의견은 다음 두 룰의 교차 검증으로 산출됩니다:

① 상승여력 룰 — Base 시나리오 목표주가 대비 현재가 상승여력 ≥ +15% BUY / -10%~+15% HOLD / < -10% SELL
② 6-Lens 룰 (Hardik Framework) — 6개 Lens의 가중 점수 ≥ 8.0 BUY / 6.0~7.9 HOLD / < 6.0 AVOID

Hardik Kawale 원문 (블록 J 흡수의 배경)
"AI today is powerful, but it has two flaws when used for investing: 1) hallucinates missing data 2) most prompts too casual. So I built a prompt that forces AI to behave like a world-class Equity Research Analyst operating in strict research-safe mode."

v11.0은 이 철학을 흡수해 블록 J(Anti-Hallucination Protocol) + Step 7 가중치 점수로 정식 통합.
본 메타 리포트는 GIC(Global Investment Club) 학회 내부 교육 목적이며, 투자 권유가 아닙니다. 본 리포트의 분석 결과는 분석자의 독립 계산이며, AI 프롬프트는 분석 도구에 불과합니다. 투자 결정은 본인 책임이며, 본 시스템은 어떠한 손익에 대해서도 책임지지 않습니다.
