## 3분 시작 가이드

### Step 1. 본인이 무엇을 할지 정하기

| 목적 | 사용 파일 | 출력 |
|---|---|---|
| 종목 1개 정식 분석 | `GIC_기업리서치_v11.md` | HTML(10~20p) + CSV + Excel |
| 매주 종목 업데이트 | `GIC_위클리투자리포트_v11.md` | HTML(최대 5p) + CSV |
| 사전 선정 종목 비교 | `GIC_산업TopPick_v11.md` | HTML(8~12p) + CSV |

### Step 2. AI 챗봇 환경 + 웹검색 활성화 (필수)

| AI | 웹검색 활성 방법 |
|---|---|
| **Claude** (claude.ai) | 자동 (Search 도구 기본 ON) |
| **ChatGPT** (chatgpt.com) | 메시지 입력란 도구 → "웹 검색" 토글 ON |
| **Gemini** (gemini.google.com) | Search Grounding 자동 ON |
| **Perplexity** | 자체가 검색 엔진 |

### Step 3. HTML→PDF 인쇄 워크플로

1. AI 챗봇에서 프롬프트 실행 → HTML 출력 받기
2. HTML을 `.html` 파일로 저장 (텍스트 에디터 또는 브라우저 "다른 이름으로 저장")
3. 브라우저에서 파일 열기
4. **Ctrl+P** (인쇄) → "PDF로 저장" → **양식별 레이아웃**:
   - **위클리 / 기업리서치** → 레이아웃: **세로** (A4 portrait)
   - **산업 Top Pick** → 레이아웃: **가로** (A4 landscape)
5. 여백: 기본 / 배경 그래픽: ON → 저장
6. 페이지가 잘리지 않고 슬라이드별로 분리되어 출력됨 (v11.0 CSS 처리)

## v11.0 8대 핵심 원칙

1. **HTML 우선**: 모든 산출물은 HTML(A4 가로) 우선 → 부원이 브라우저 인쇄로 PDF
2. **데이터 파일 동시 출력**: 재무 표는 csv 코드블록 + (가능 시) Excel 자동 생성
3. **웹검색 의무화**: 최신 주가·시총·뉴스는 챗봇 내장 웹검색으로 갱신 + URL 첨부 필수
4. **페이지 분할 안전 CSS**: A4 가로 297×210mm + `page-break-after: always` + `break-inside: avoid`
5. **위클리 독립**: 정식 분석 없이도 단독 핵심 분석 가능 (5p 이내)
6. **Top Pick 사전 선정**: 조에서 종목 입력 → 비교·분석만 (발굴 X)
7. **프롬프트 메타 공개**: 사용한 프롬프트 기법(블록 A~J, 6-Lens 등)을 본문 끝 + 별도 HTML로 설명
8. **외부 의존 0**: API 키·웹앱·MCP 모두 불필요. 챗봇 복붙만으로 동작
