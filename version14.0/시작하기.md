# 🚀 시작하기 (5분 컷)

처음이세요? 이 순서만 그대로 따라 하세요. **유료 결제 0원, OpenDART 무료 키 하나면 끝.**

## 0. 준비물 (전부 무료)

- Windows + 인터넷
- Python 3.11 이상 → [python.org](https://www.python.org/downloads/) 에서 설치 (설치 시 **"Add Python to PATH" 체크**)
- OpenDART 무료 API 키 → https://opendart.fss.or.kr/ 가입 → "인증키 신청" → 메일로 받음 (1분)

## 1. 설치 (딱 한 번)

PowerShell을 열고:

```powershell
cd "이_폴더_경로\version14.0"
python -m pip install -e .
```

> `pip` 에러가 나면 `python -m pip install --upgrade pip` 먼저.

## 2. 키 한 번 넣기 (이 창에서만 유효, 안전)

```powershell
$env:OPENDART_API_KEY="여기에_본인_OpenDART_키_붙여넣기"
```

> 키는 코드/결과물 어디에도 저장되지 않습니다. 절대 파일에 적지 마세요.

## 3. 분석할 기업 정하기

`input\company_request.yaml` 을 메모장으로 열어 **종목코드/기업명/피어**만 바꾸세요. (나머지는 그대로 둬도 됨)

## 4. 실행

```powershell
python -m gic_v14.cli run input\company_request.yaml --output-dir output
```

## 5. 결과 보기 → `output\<run_id>\deliverables\`

| 파일 | 무엇 | 어떻게 |
|---|---|---|
| `report.html` | GIC 양식 리포트(데이터·차트 자동 채움) | **더블클릭 → 크롬에서 열기**. 인쇄(Ctrl+P) → PDF 저장 |
| `prompt_pack.md` | 단계별 분석 프롬프트(실제 수치 주입됨) | 복사 → ChatGPT/Claude/Gemini에 붙여넣어 `[작성 필요]` 칸 채우기 |
| `..\audit\qa_report.md` | 자동/수동 구분, 미수집 항목 | 검토용 |

## 6. 마무리

`report.html`의 주황색 `[작성 필요]` 칸을 prompt_pack 결과로 채우면 리포트 완성!

## ❓ 자주 막히는 곳

- **표/로고가 안 보여요** → `report.html`을 미리보기 말고 **크롬/엣지로 직접** 여세요. (이미지는 같은 폴더에 함께 생성됨)
- **`OPENDART_API_KEY is required`** → 2번 키 넣기를 안 했거나 창을 새로 열었어요. 다시 넣으세요.
- **키 없이 구경만** → `python -m gic_v14.cli run tests\fixtures\... ` 대신 아래 데모 참고.

자세한 설명은 [README.md](README.md), 자동화(스케줄·메일)는 [docs/AGENT_AUTOMATION_GUIDE.md](../version13.0/docs/AGENT_AUTOMATION_GUIDE.md) 참고.
