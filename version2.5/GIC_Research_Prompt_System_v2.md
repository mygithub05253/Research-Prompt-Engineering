# GIC 기업 리서치 리포트 — 모듈형 프롬프트 시스템

> **버전**: v1.0  
> **작성일**: 2026.04.30  
> **대상**: Gachon Investment Club (GIC) 리서치 팀  
> **호환 AI 도구**: Claude, ChatGPT(Codex), Gemini, Perplexity 등

---

## 사용 방법

이 문서는 **8단계 모듈형 프롬프트**로 구성되어 있습니다.  
각 Step의 프롬프트를 AI 도구에 복사-붙여넣기하고, `[대괄호]` 안의 내용을 본인의 분석 대상에 맞게 수정하세요.  
각 단계에서 데이터 파일을 첨부해야 하는 경우 📎 표시로 안내합니다.

**실행 순서:**  
Step 0 → Step 1 → Step 2 → Step 3 → Step 4 → Step 5 → Step 6 → Step 7 → Step 8

## Step 0: 초기 설정 (기업 선정 & 분석 프레임 설정)

```
당신은 한국 대학교 투자 동아리 소속 기업 리서치 애널리스트입니다.
지금부터 기업 리서치 보고서를 단계별로 작성할 것입니다.

아래 정보를 기반으로 분석 프레임을 설정해주세요.

■ 분석 대상 기업: [기업명] ([종목코드])
■ 섹터 분류: [아래 9개 중 택1]
  - 인프라 / 금융 / 반도체 / 로봇 / 조선 / 방산 / 소비재 / 바이오 / AI
■ 분석 기준일: [YYYY.MM.DD]
■ 분석 관점 (택1 또는 복수 선택):
  - 성장주 관점 / 가치주 관점 / 턴어라운드 / 배당 매력 / 테마·모멘텀
■ 투자포인트 개수: [숫자] 개 (기본값: 3)
■ 투자리스크 개수: [숫자] 개 (기본값: 2)

아래 내용을 정리해서 출력해주세요:
1. 기업 개요 한 줄 요약 (업종, 주력 사업, 시장 내 위치)
2. 이 섹터에서 현재 주목해야 할 핵심 키워드 3~5개
3. 분석 시 집중적으로 살펴볼 재무 지표 추천 (해당 섹터 특성 반영)
4. 이후 단계에서 필요한 데이터 목록 체크리스트

이 설정은 이후 모든 단계에서 일관되게 적용됩니다.
```

## Step 1: 경쟁사 매핑 & CSV 생성

```
[Step 0에서 설정한 기업명/섹터 정보를 여기에 붙여넣으세요]

위 기업이 속한 섹터에서 사업 모델 유사성을 기준으로 경쟁사를 매핑해주세요.

■ 매핑 기준:
  - 동일 섹터 내 사업 모델이 유사한 기업 (국내 + 글로벌)
  - 매출 구성, 핵심 기술/제품, 고객군, 밸류체인 내 포지션 기준
  - 국내 기업 최소 3개, 글로벌 기업 최소 3개 포함

■ 출력 형식: 아래 컬럼을 포함한 표(CSV 호환) 형태로 정리
  | 기업명 | 종목코드/티커 | 국가 | 시가총액 | 주력 사업 | 사업 유사도(상/중/하) | 비고 |

■ 추가 요청:
  1. 각 경쟁사가 왜 유사한지 한 줄 근거를 '비고' 컬럼에 작성
  2. 이 경쟁사 리스트를 CSV 파일로 다운로드할 수 있도록 코드 또는 파일 생성
  3. 경쟁사 간 시가총액 규모 비교 차트를 HTML로 생성

※ 웹 검색을 활용하여 최신 시가총액과 사업 현황을 반영해주세요.
※ 경쟁사 리스트를 확인한 후, 추가/삭제하고 싶은 기업이 있으면 알려주세요.
```

## Step 2: 재무 데이터 분석 & 시각화

> 📎 **이 단계에서 첨부할 파일:**  
> DART 전자공시시스템에서 다운로드한 재무제표 파일 (엑셀 또는 PDF)  
> - 재무상태표 (최근 3~5개년)  
> - 손익계산서 (최근 3~5개년)  
> - 현금흐름표 (최근 3~5개년)  
> - 분기별 실적 데이터 (있으면 첨부)

```
첨부한 파일은 [기업명]의 DART 전자공시 재무제표입니다.
이 데이터를 분석하여 아래 작업을 수행해주세요.

■ 1. 재무제표 파싱 & 정리
  - 첨부 파일에서 재무 데이터를 추출하여 정리된 표로 출력
  - 연도별 주요 항목: 매출액, 영업이익, 당기순이익, 자산총계, 부채총계, 자본총계, 영업활동현금흐름
  - 단위 통일 (억원 또는 조원)

■ 2. 핵심 재무 지표 계산
  [수익성]
  - 매출총이익률, 영업이익률, 순이익률, ROE, ROA, ROIC

  [성장성]
  - 매출액 증가율, 영업이익 증가율, 순이익 증가율 (YoY)

  [안정성]
  - 부채비율, 유동비율, 이자보상배율, 순차입금비율

  [효율성]
  - 총자산회전율, 재고자산회전율, 매출채권회전율

  [밸류에이션 기초]
  - EPS, BPS, PER, PBR, EV/EBITDA, 배당수익률

■ 3. 시각화 (HTML 차트 생성)
  아래 차트를 HTML 파일로 생성해주세요:
  ① 연도별 매출액·영업이익·순이익 추이 (막대+선 혼합 차트)
  ② 수익성 지표 추이 (영업이익률, 순이익률, ROE — 선 차트)
  ③ 성장성 지표 추이 (YoY 증가율 — 막대 차트)
  ④ 재무 안정성 대시보드 (부채비율, 유동비율 추이)
  ⑤ [Step 1에서 선정한 주요 경쟁사 3~4개]와의 핵심 지표 비교 테이블

■ 4. 분석 코멘트
  각 지표 카테고리별로 2~3문장의 분석 코멘트를 작성해주세요.
  특히 아래 질문에 답변하는 방향으로:
  - 이 기업의 수익성은 개선 추세인가, 악화 추세인가?
  - 재무 안정성에 우려 사항이 있는가?
  - 동종 업계 대비 어떤 지표에서 강점/약점을 보이는가?

※ 데이터에서 이상치나 누락이 발견되면 반드시 알려주세요.
```

## Step 3: 산업분석 작성

```
[Step 0의 기업명/섹터 정보를 붙여넣으세요]

위 기업이 속한 [섹터명] 섹터의 산업분석을 작성해주세요.
웹 검색을 통해 최신 정보를 반드시 반영해주세요.

■ 작성 항목:

1. 시장 규모 & 성장 전망
   - 글로벌/국내 시장 규모 (금액 기준)
   - 향후 3~5년 CAGR 전망
   - 시장 성장의 핵심 드라이버

2. 산업 구조 & 경쟁 환경
   - 밸류체인 내 분석 대상 기업의 포지션
   - 시장 점유율 현황 (가능한 경우 수치 포함)
   - 진입장벽 수준과 주요 요인
   - 경쟁 강도 및 최근 변화

3. 주요 산업 트렌드 (3~5개)
   - 각 트렌드가 분석 대상 기업에 미치는 영향 (긍정/부정)
   - 근거 자료 출처 명시

4. 규제 & 정책 환경
   - 국내외 주요 규제 동향
   - 정부 정책 수혜/리스크 요인

■ 작성 톤: 증권사 리서치 리포트 스타일 (간결하고 논리적, 데이터 기반)
■ 분량: A4 1~1.5페이지 분량
■ 관련 시장 데이터 차트가 있다면 HTML로 생성해주세요.
```

## Step 4: 기업분석 작성

> 📎 **이 단계에서 추가 첨부 가능:**  
> 사업보고서, IR 자료, 실적발표 프레젠테이션 등 (선택사항)

```
[Step 0의 기업명/섹터 정보를 붙여넣으세요]
[Step 2의 재무 분석 결과 요약을 붙여넣으세요]

위 재무 분석 결과와 웹 검색을 활용하여 기업분석 섹션을 작성해주세요.

■ 작성 항목:

1. 사업 구조 & 매출 구성
   - 사업부별/제품별 매출 비중
   - 핵심 수익원과 성장 동력
   - 주요 고객사 및 매출 집중도

2. 최근 실적 분석
   - 가장 최근 분기/연간 실적 요약
   - 전년 대비 주요 변동 사항과 원인
   - 컨센서스 대비 서프라이즈/미스 여부

3. 경영진 & 지배구조
   - 경영진 핵심 역량 및 트랙레코드
   - 지배구조 리스크 (있는 경우)
   - 최근 경영전략 방향

4. 주주환원 정책
   - 배당 정책 (배당금, 배당성향, 자사주 매입 등)
   - 향후 주주환원 확대 가능성

■ 작성 톤: 증권사 리서치 리포트 스타일
■ 분량: A4 1~1.5페이지 분량
■ 실적 추이 차트는 Step 2에서 생성한 것을 참조
```

## Step 5: 투자포인트 & 투자리스크 도출

```
[Step 0의 기업명/섹터/투자포인트·리스크 개수 설정을 붙여넣으세요]
[Step 2 재무 분석 결과 요약을 붙여넣으세요]
[Step 3 산업분석 결과 요약을 붙여넣으세요]
[Step 4 기업분석 결과 요약을 붙여넣으세요]

위 분석 결과를 종합하여 투자포인트와 투자리스크를 도출해주세요.

■ 투자포인트 [N]개 — 각 포인트별 아래 형식:
  - 소제목 (핵심 메시지를 한 문장으로)
  - 좌측 키워드 (리포트 사이드바에 들어갈 2줄 이내 요약)
  - 본문 (4~6문장, 구체적 수치와 논리적 근거 포함)
  - 근거 데이터/출처

■ 투자리스크 [M]개 — 각 포인트별 아래 형식:
  - 소제목
  - 좌측 키워드
  - 본문 (4~6문장, 리스크의 발생 조건과 영향 범위 명시)
  - 리스크 완화 요인 (있는 경우)

■ 작성 원칙:
  1. 투자포인트는 '왜 지금 사야 하는가'에 대한 명확한 답변이어야 함
  2. 투자리스크는 '어떤 시나리오에서 투자가 실패할 수 있는가'를 정직하게 제시
  3. 각 포인트는 서로 겹치지 않는 독립적인 논거여야 함
  4. 추상적 서술 금지 — 반드시 수치, 시점, 조건을 명시
  5. 웹 검색으로 최신 뉴스/보고서를 반영하여 근거 강화

※ 각 포인트에 대해 신뢰도 수준(상/중/하)도 함께 표시해주세요.
```

## Step 6: 밸류에이션

```
[Step 0의 기업명/종목코드를 붙여넣으세요]
[Step 1의 경쟁사(Peer Group) 리스트를 붙여넣으세요]
[Step 2의 재무 지표 (EPS, BPS, PER, PBR, EV/EBITDA 등)를 붙여넣으세요]

위 데이터를 기반으로 밸류에이션을 수행해주세요.

■ 핵심 방법론: 예상 EPS × Peer Group 평균 PER
  1. 분석 대상 기업의 향후 12개월 예상 EPS 추정
     - 추정 근거를 반드시 명시 (매출 성장률 가정, 마진 가정 등)
  2. Peer Group의 현재 PER 정리 (표 형태)
  3. Peer 평균/중앙값 PER 산출
  4. 적정 PER 범위 설정 (할인/할증 사유 포함)
  5. 목표주가 = 예상 EPS × 적정 PER → 산출

■ 보조 방법론 (가능한 범위 내 수행):

  [상대가치 비교]
  - PER, PBR, EV/EBITDA 기준 Peer 대비 할인/할증률 분석

  [DCF (잉여현금흐름 할인)]
  - FCFF 추정 (향후 5년)
  - WACC 산출 (가능한 경우)
  - 영구성장률(Terminal Growth Rate) 가정
  - DCF 기반 적정 주가 산출
  - 민감도 분석 (WACC ± 0.5%p, 영구성장률 ± 0.5%p)

  [SOTP (부문별 가치합산)]
  - 사업부별 매출/이익 분리
  - 각 사업부에 적합한 Peer 멀티플 적용
  - 사업부별 가치 합산 → 적정 기업가치 산출

■ 최종 출력:
  1. 방법론별 적정주가 요약 테이블
     | 방법론 | 적정주가 | 현재주가 대비 괴리율 |
  2. 목표주가 최종 결정 (근거 1~2문장)
  3. 투자의견: BUY / HOLD / SELL (근거 포함)
  4. 밸류에이션 민감도 분석 차트 (HTML 생성)

※ 웹 검색을 활용하여 Peer Group의 최신 밸류에이션 멀티플을 조회해주세요.
※ 모든 가정(assumption)은 명시적으로 기술하고, 보수적/기본/낙관 시나리오를 제시해주세요.
```

## Step 7: 검토 & 수정

```
지금까지 작성한 리서치 리포트의 각 섹션을 아래 순서대로 한눈에 정리해서 보여주세요.

■ 보여줄 항목:
  1. [커버] 기업명, 종목코드, 목표주가, 현재주가, 상승여력, 투자의견
  2. [산업분석] 핵심 요약 3줄
  3. [기업분석] 핵심 요약 3줄
  4. [투자포인트] 각 포인트 소제목 + 한 줄 요약
  5. [투자리스크] 각 포인트 소제목 + 한 줄 요약
  6. [밸류에이션] 목표주가, 사용 방법론, 핵심 가정

■ 체크리스트 (아래 항목을 자체 점검하고 결과를 표시):
  □ 목표주가와 투자의견이 논리적으로 일관되는가?
  □ 투자포인트와 리스크가 서로 중복되지 않는가?
  □ 모든 핵심 수치에 출처 또는 추정 근거가 있는가?
  □ 산업분석 → 기업분석 → 투자포인트의 논리 흐름이 자연스러운가?
```

## 부록 C: 밸류에이션 방법론 상세 가이드

### 1. 핵심: 예상 EPS × Peer Group PER

```
적정주가 = Forward EPS × 적정 PER

- Forward EPS: 향후 12개월 예상 주당순이익
  → 추정 방법: (예상 매출 × 예상 순이익률) ÷ 발행주식수
  
- 적정 PER: Peer Group 평균 PER 기반
  → 할증 사유: 시장 지배력, 성장률 프리미엄, 기술 격차
  → 할인 사유: 지배구조 리스크, 변동성, 유동성 할인
```

### 2. 상대가치 비교

```
분석 대상 기업의 PER/PBR/EV/EBITDA를 Peer와 비교
→ 현재 할인율/할증률 산출
→ 역사적 평균 대비 현 수준 판단 (과거 5년 밴드 활용)
```

### 3. DCF (Discounted Cash Flow)

```
기업가치 = Σ(FCFF_t / (1+WACC)^t) + Terminal Value / (1+WACC)^n

- FCFF = 영업이익×(1-세율) + 감가상각비 - CAPEX - 운전자본 증감
- WACC = E/(E+D)×Ke + D/(E+D)×Kd×(1-t)
- Terminal Value = FCFF_(n+1) / (WACC - g)
- 민감도 분석: WACC(±0.5~1.0%p) × 영구성장률(±0.5%p) 매트릭스
```

### 4. SOTP (Sum of the Parts)

```
기업가치 = Σ(사업부별 가치) - 순차입금

- 각 사업부: 해당 사업부와 가장 유사한 Pure-play Peer의 멀티플 적용
- 할인율: 지주사 할인, 비핵심 사업부 할인 등 반영
```

| 6 | 투자포인트 (4~6) | 섹션바 + 본문 Table(2×2) ×3 (필요 없으면 빈칸 처리) |
| 7 | 투자리스크 | 섹션바 + 본문 Table(2×2) ×2 + 하단 이미지/차트 영역 |

| 9 | 밸류에이션 2 + 최종의견 | 재무 Table(6×9) + 최종의견 텍스트 + Compliance Notice |

### 핵심 본문 Table 구조 (2×2 패턴)

산업분석, 기업분석, 투자포인트, 투자리스크의 본문은 모두 동일한 **2행 × 2열 Table** 구조:

```
┌──────────────┬────────────────────────────────────┐
│              │ Row 0, Col 1: 소제목               │
│  (빈칸)      │ "Point1. 가나다라" 등              │
├──────────────┼────────────────────────────────────┤
│ Row 1, Col 0 │ Row 1, Col 1: 본문                 │
│ 좌측 키워드  │ 4~6문장 분석 내용                   │
│ (2~3줄)      │                                    │
└──────────────┴────────────────────────────────────┘
```

**교체 방법:**
```python
table = slide.shapes[shape_index].table
set_cell_text(table, 0, 1, '소제목 텍스트')      # 제목
set_cell_text(table, 1, 0, '좌측\n키워드')        # 키워드 (줄바꿈 가능)
set_cell_text(table, 1, 1, '본문 내용 4~6문장')   # 본문
```
