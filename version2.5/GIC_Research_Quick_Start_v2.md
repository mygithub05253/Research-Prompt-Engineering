## 5분 만에 시작하기

### 1단계: AI 도구를 열고 Step 0 프롬프트를 복사-붙여넣기

```
당신은 한국 대학교 투자 동아리 소속 기업 리서치 애널리스트입니다.
지금부터 기업 리서치 보고서를 단계별로 작성할 것입니다.

■ 분석 대상 기업: [기업명] ([종목코드])
■ 섹터 분류: [인프라/금융/반도체/로봇/조선/방산/소비재/바이오/AI 중 택1]
■ 분석 기준일: [YYYY.MM.DD]
■ 분석 관점: [성장주/가치주/턴어라운드/배당/테마 중 택1]
■ 투자포인트 개수: [숫자]개
■ 투자리스크 개수: [숫자]개

기업 개요, 섹터 핵심 키워드, 집중 분석 지표, 필요 데이터 목록을 정리해주세요.
```

`[대괄호]` 안의 내용만 본인 분석 대상에 맞게 수정하면 됩니다.

### 2단계: AI 응답을 확인하고 다음 Step 프롬프트로 이동

- 각 Step의 결과를 확인하고, 다음 Step 프롬프트에 이전 결과를 붙여넣습니다.
- Step 2에서는 DART 재무 데이터 파일을 첨부합니다.
- Step 8에서는 GIC 머스타치 양식 PPTX (GIC_template_v3.pptx)를 첨부합니다. 양식 안의 {{토큰}}이 자동 교체됩니다.

### 3단계: Step 7에서 전체를 검토하고 수정 요청

- "투자포인트 2번 근거 보강해줘"
- "목표주가에서 PER 25배로 변경해줘"
- 만족하면 "확정"이라고 입력 → Step 8로 이동

### 4단계: Step 8에서 최종 리포트 생성

Step 8은 양식의 {{토큰}}을 실제 데이터로 교체하고 차트를 삽입합니다.
- 8-A: 웹에서 주가 차트, 재무 차트 등을 이미지로 수집
- 8-B: 양식 파일을 python-pptx/python-docx로 열어서 내용만 교체
- **디자인은 절대 변경하지 않고, 텍스트와 이미지만 교체합니다**

## 각 Step 요약 (1줄 설명)

| Step | 이름 | 하는 일 | 소요 시간 |
|------|------|---------|----------|
| 0 | 초기 설정 | 기업명, 섹터, 분석 관점 입력 | 2분 |
| 1 | 경쟁사 매핑 | 동종 업계 경쟁사 리스트 + CSV | 5분 |
| 2 | 재무 분석 | 📎 DART 데이터 첨부 → 지표 계산 + 차트 | 10분 |
| 3 | 산업분석 | 시장 규모, 트렌드, 경쟁 구도 작성 | 5분 |
| 4 | 기업분석 | 실적, 사업구조, 주주환원 작성 | 5분 |
| 5 | 투자포인트/리스크 | 핵심 투자 논거 도출 | 5분 |
| 6 | 밸류에이션 | 목표주가 산출 (EPS×PER + DCF 등) | 10분 |
| 7 | 검토 & 수정 | 전체 리뷰, 수정 요청, 확정 | 5~15분 |
| 8 | 최종 출력 | 📎 머스타치 양식 첨부 → {{토큰}} 교체 → 차트 삽입 → PPTX+PDF | 10분 |

**총 예상 소요 시간: 55분~1시간 10분**

> **⚠️ Step 8 핵심 주의사항:**  
> 양식 파일은 반드시 **편집 가능한 형식(PPTX 또는 DOCX)**으로 첨부하세요.  
> PDF만 있다면 먼저 PPTX/DOCX로 변환 후 첨부해야 합니다.  
> AI가 디자인을 새로 만들지 않고, 기존 양식을 열어서 내용만 교체합니다.

## AI 도구별 추천 사용법

### Claude
- 파일 첨부(PDF/Excel)가 자연스러움 → Step 2, 8에서 가장 편리
- 긴 대화 맥락 유지에 강함 → 하나의 대화에서 Step 0~8 연속 실행 가능

### ChatGPT (Codex)
- Code Interpreter로 Python 차트 직접 생성 → "matplotlib으로 차트 그려줘" 추가
- 파일 업로드 지원 → Step 2에서 엑셀 분석 가능

### Gemini
- 웹 검색이 기본 통합 → Step 1, 3에서 최신 데이터 수집에 강점
- Deep Research 모드 활용 시 산업분석 품질 향상

### Perplexity
- 출처 자동 인용 → Step 3 산업분석, Step 5 투자포인트 근거 확보에 유용
- 실시간 검색 특화 → 경쟁사 최신 뉴스, 밸류에이션 멀티플 조회

## DART 재무 데이터 다운로드 (3분 가이드)

1. **dart.fss.or.kr** 접속
2. 검색창에 기업명 입력 → 검색
3. 좌측 메뉴 "정기공시" 클릭
4. 가장 최근 **사업보고서** (또는 분기보고서) 클릭
5. 목차에서 **"III. 재무에 관한 사항"** 찾기
6. 재무상태표, 손익계산서, 현금흐름표를 **엑셀 다운로드** (또는 PDF 저장)
7. 최근 3~5개년 데이터가 필요 → 과거 보고서도 추가 다운로드

> **팁**: CompanyGuide(comp.fnguide.com)에서 "재무분석" 탭을 이용하면 5개년 데이터를 한 번에 엑셀로 받을 수 있습니다.

## 자주 묻는 질문

**Q: 꼭 순서대로 해야 하나요?**  
A: 네, Step 0~6은 순서대로 진행하는 것을 권장합니다. 각 단계의 출력이 다음 단계의 입력으로 사용되기 때문입니다. 다만 Step 3(산업분석)과 Step 4(기업분석)는 순서를 바꿔도 됩니다.

**Q: 한 번의 대화에서 다 끝내야 하나요?**  
A: 아닙니다. 각 Step의 결과를 따로 저장해두고, 새 대화에서 이전 결과를 붙여넣어도 됩니다. 다만 하나의 대화에서 연속으로 하면 AI가 맥락을 더 잘 유지합니다.

**Q: 재무 데이터 엑셀 형식이 정해져 있나요?**  
A: 아닙니다. DART에서 다운받은 그대로 첨부하면 AI가 파싱합니다. CompanyGuide, FnGuide 등 다른 소스의 엑셀도 사용 가능합니다.

**Q: 리포트 양식 파일이 없으면 어떻게 하나요?**  
A: Step 8은 양식 파일(PPTX 또는 DOCX)이 있어야 디자인을 그대로 유지할 수 있습니다. PDF 양식만 있는 경우, AI가 디자인을 참고하여 새로 만들지만 원본과 차이가 생길 수 있으므로, 가능하면 편집 가능한 PPTX/DOCX 원본을 준비하세요.

**Q: Step 8에서 차트 이미지를 직접 준비해야 하나요?**  
A: Claude나 ChatGPT처럼 코드 실행이 가능한 AI 도구는 웹 크롤링으로 차트를 자동 수집합니다. 코드 실행이 안 되는 도구(Gemini, Perplexity 등)를 사용하는 경우, AI가 캡처할 URL과 영역을 안내해주므로 직접 캡처하면 됩니다.

**Q: 프롬프트를 수정해도 되나요?**  
A: 당연합니다. 각 Step의 "작성 항목"을 추가/삭제하거나, 분량을 조절하거나, 분석 깊이를 바꿀 수 있습니다. 본인의 분석 스타일에 맞게 커스터마이징하세요.
