## 검증 룰 (Skill stages/1_intake.md에서 자동 체크)

1. **필수 변수 누락**: 종목명·종목코드·투자의견·revision 3개·재무지표 표 5행 → 누락 시 ERROR
2. **글자 수 위반**: ≤N자 룰 위반 시 WARNING + 자동 절단 옵션 제안
3. **자리표시자 미채움**: `[###]`, `[기업명]` 등 그대로 남아있으면 ERROR
4. **이미지 경로 무효**: 파일 경로가 존재하지 않으면 WARNING (해당 슬라이드 빈 영역으로 출력)
5. **표 행 수 불일치**: 재무지표 표는 정확히 5행 필요. 부족 시 빈 행 자동 추가
