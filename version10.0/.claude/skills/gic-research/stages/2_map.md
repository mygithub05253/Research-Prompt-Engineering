# Stage 2 — Map (슬라이드별 변수 매핑)

## 입력
Stage 1 검증 통과 yaml 객체

## 처리
1. yaml의 모든 키를 columns.md 매핑 표에 따라 슬라이드 1~9 텍스트 박스/표/이미지에 매핑
2. 미리보기 생성: 슬라이드별 핵심 텍스트 요약

## 출력 (부원 미리보기)
```
[Map 미리보기]

슬라이드 1 — 표지
  종목명: 삼성전기 (009150)
  투자의견: BUY (매수)
  목표주가: 195,000원 / 현재가: 152,500원 / 상승여력: +27.9%
  Check Point: AI 서버 MLCC 1위, 멀티플 디스카운트 해소
  Point 1: AI 서버 MLCC 점유율 확대 (2024 33% → 2026E 41%)
  Point 2: 카메라 모듈 디카메라화로 ASP 상승
  Point 3: 멀티플 디스카운트 해소 - 글로벌 피어 대비 -25% PER

슬라이드 2 — CONTENTS (5섹션)
슬라이드 3 — 산업분석: 글로벌 AI 서버 시장 5년 CAGR 38%
슬라이드 4 — 기업분석: MLCC + 카메라 모듈 양대 사업 구조
슬라이드 5·6 — 투자포인트: Point 1·2·3
슬라이드 7 — 투자리스크: 중국 MLCC 저가 공세 / AI 서버 수요 급랭
슬라이드 8 — 밸류에이션 (1): 재무지표 표 + 차트 4개
슬라이드 9 — 밸류에이션 (2) + Compliance Notice

[승인하시려면 "다음" 입력, 수정하시려면 어떤 슬라이드를 수정할지 알려주세요]
```

## 게이트
- 부원 "다음" 입력 → Stage 3 진행
- 부원 "수정" 요청 → 해당 yaml 키 수정 → Stage 1 재실행
