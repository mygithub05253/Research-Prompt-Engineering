# gic-toppick rules.md

## 디자인
- 표지·결론 다크 베이스, 본문 라이트 베이스 (다크/라이트 교차로 페이스 메이커)
- design-guide_v10 §3 토큰 적용
- 시그니처: 슬라이드당 1~2개

## 슬라이드 수 조정
- Top Pick 1개: 7슬라이드 (슬라이드 5 생략)
- Top Pick 2개: 8슬라이드 (기본)
- Top Pick 3개+: ERROR (TopPick은 결정적 추천 — 산업 분석 별도 권장)

## 후보 종목 개수
- 5~10개 1차 스크리닝
- 10개 초과: 2단계 스크리닝 권장 (10~20개 → 5~7개 → Top 1~2)

## PPTX 호환성
gic-weekly와 동일 룰 적용
