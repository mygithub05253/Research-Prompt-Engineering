## 산업 Top Pick 사용 워크플로

### Top Pick 1개 vs 2개 선택 기준
- **1개**: 산업이 명확한 1위 종목을 가질 때 (반도체의 삼성전자 같은)
- **2개**: 산업 내 분기점에서 둘 다 노릴 가치 있을 때 (조선의 HD현대重 vs 삼성중공업)
- **3개 이상**: 산업 분석으로 분리 권장 (Top Pick은 결정적 추천이어야 함)

### Top Pick과 정식 기업리서치의 차이
| 구분 | Top Pick | 기업리서치 |
|---|---|---|
| 흐름 | 산업 → 종목 (top-down) | 기업 → 산업 (bottom-up) |
| 깊이 | 압축 (Phase 4) | 정식 9단계 풀 |
| 빈도 | 산업 변곡점마다 | 종목당 1회 풀 + 위클리 |
| 용도 | 발견·탐색 | 정식 발표 |

### 후보 종목이 10개를 넘으면
1차 스크리닝 (10~20개) → 점수 상위 5~7개 압축
2차 스크리닝 (5~7개) → 정밀 점수 + Red Team 통과 가능성 → Top 1~2

## AI별 Top Pick 활용 팁 (본문 [점대괄호] 분기와 동일)

### Claude
- **Artifacts**: Phase 2 스크리닝 표를 인터랙티브 정렬·필터로
- **PDF 다중 첨부**: 후보 5~10개 사업보고서 200K 토큰 컨텍스트
- **회의주의**: Phase 3 Top Pick 선정 시 자체 Red Team 가능성 사전 체크

### ChatGPT
- **웹 브라우징 풀가동**: Phase 1 산업 리포트 5건 자동 검색
- **Code Interpreter**: Phase 2 스크리닝 점수 자동 계산 + Excel 다운로드
- **GPTs Memory**: 매주 같은 산업 추적 시 일관성 유지

### Gemini
- **Search Grounding**: 산업 동향·정책·M&A 자동 검색 (가장 강력)
- **2M 토큰**: 후보 5~10개 사업보고서 PDF 통째 첨부
- **유튜브 분석**: 산업 컨퍼런스·IR 영상 트랜스크립트 (Advanced)
- **Google Sheets 연동**: 스크리닝 점수 시트 자동 생성 (Advanced)
