## 1. 이 시스템은 무엇인가

가천대학교 GIC(Global Investment Club) 4기에서 만든 **AI 챗봇용 기업 리서치 프롬프트 시스템**입니다.

### 핵심 사양
- **외부 의존 0**: API 키 / 웹앱 / MCP 서버 / Python 코드 모두 불필요
- **챗봇 복붙 100%**: Claude / ChatGPT / Gemini / Perplexity 모든 챗봇에서 동작
- **3종 산출물**: ① 정식 기업리서치 ② 위클리 투자리포트 ③ 산업 Top Pick
- **9단계 모듈형**: 단계별로 검토하며 발표용 PPT까지 한 번에

### 무엇을 만들 수 있나
- 증권사 PB급 정식 기업 리서치 리포트 (7페이지 PPT)
- 매주 신속 갱신 위클리 (4페이지 PPT)
- 산업 진단 + 최선호 종목 추천 (5~8 슬라이드)

### 무엇을 만들 수 없나
- 실시간 자동 매매 시스템
- DART/KRX API를 직접 호출하는 자동화 프로그램
- 학회 발표 양식 그대로의 PPT 파일 자동 생성

## 5. 자동화 부록 — 티어 1·2

> v9.0은 자동화에 의존하지 않습니다. 티어 1(.md 복붙)이 베이스입니다.
> 티어 2는 익숙한 부원의 추가 옵션. 티어 3·4는 v10에서 다룰 예정.

### 티어 1 — .md 챗봇 복붙 (필수, 100% 부원)
- **필요 환경**: AI 챗봇 1개 (무료도 가능)
- **진입 시간**: 0
- **결과물**: 마크다운 표 + Mermaid + 텍스트 복붙용 슬라이드 본문
- **사용 파일**: `GIC_기업리서치_v9.md` 정식 9단계
- **이게 베이스다.** 모든 부원이 여기서 시작·완료 가능.

### 티어 2 — Excel 매크로 + 학회 양식 (선택, 익숙한 부원)
- **필요 환경**: Microsoft Excel + VBA 활성
- **진입 시간**: 30분 (양식 익히기)
- **결과물**: 5개년 IS/BS/CF 모델 + 시나리오 매트릭스 + 자동 그래프
- **사용 시점**: AI가 출력한 마크다운 표 → Excel 양식에 붙여넣기 → 자동 계산

#### 5.1 Excel 양식 구성 (학회 측에서 만들 권장)

| 시트 | 내용 | 자동화 |
|---|---|---|
| Sheet 1: Assumptions | Bear/Base/Bull 가정표 | 입력 셀 색상 구분 |
| Sheet 2: IS | 5개년 손익계산서 | YoY 성장률 자동 |
| Sheet 3: BS | 5개년 재무상태표 | Balance Check 수식 |
| Sheet 4: CF | 5개년 현금흐름표 | FCF 자동 계산 |
| Sheet 5: Valuation | 4종 밸류에이션 | 목표주가 자동 |
| Sheet 6: Scenario | 시나리오 매트릭스 | EPS × PER 매트릭스 |
| Sheet 7: Sensitivity | 민감도 분석 | ±15% 자동 |
| Sheet 8: Summary | 1페이지 요약 | 다른 시트 참조 |

#### 5.2 워크플로 (티어 1 + 티어 2 결합)

1. AI 챗봇에서 정식 9단계 통과 (티어 1)
2. Step 6의 5개년 표 마크다운 → Excel Sheet 2로 복붙
3. Step 6의 가정표 → Sheet 1로 복붙
4. Excel이 자동으로 IS/BS/CF/밸류에이션 산출
5. 가정 변경 시 즉시 재계산 (시나리오 분석 가능)
6. Excel 결과 다시 마크다운으로 변환 → Step 8 슬라이드 복붙

#### 5.3 Excel 양식 만들기 — 학회 차원 표준화 권장

학회에서 1회 만들고 매년 재사용:
- 색상 코딩: 파란색=입력, 검정=수식, 녹색=참조 (전통)
- Balance Check 수식: `=IF(자산총계 = 부채+자본, "OK", "ERROR")`
- 목표주가 = `Forward EPS × 적정 PER`
- 시나리오는 Data Table 기능 활용

## 6. 자주 묻는 질문 (FAQ)

### Q1. AI 챗봇 어떤 걸 써야 하나?
A. 첫 분석은 **Claude 무료** 또는 **ChatGPT Plus**. 익숙해지면 본인 선호로. 출처 필수면 **Gemini Search Grounding**, 회의주의 강점은 **Claude**, 자동 코드 실행은 **ChatGPT Code Interpreter**.

### Q2. 무료 AI에서도 정식 9단계 가능?
A. 가능. 단 Claude 무료는 일일 메시지 한도, ChatGPT 무료는 GPT-4o-mini 제한 있음. 분기 1~2회 정식 분석은 무료로 충분.

### Q3. 사업보고서 PDF 100페이지 넘는데 AI가 다 읽나?
A.
- **Claude**: 200K 토큰 → 100페이지+ OK
- **Gemini**: 2M 토큰 → 100페이지+ OK
- **ChatGPT**: 50~80페이지 안전선 — III장 재무·II장 사업 부분만 잘라서 첨부 권장

### Q4. 코드블록이 잘려서 나와요
A. v9.0은 4중 백틱(```` ``` ` ````) 사용으로 잘림 0건 설계. 만약 잘리면 (1) 마크다운 뷰어가 4중 백틱 미지원이거나 (2) AI가 응답 토큰 한도에 걸린 것. 후자는 단계 분할로 해결.

### Q5. 6-Lens 가중 점수와 v8.0 상승여력 룰이 충돌해요
A. Step 7-3에서 두 룰이 충돌할 때 사유를 명시하라고 했습니다. 일반적으로:
- 두 룰 일치 → 강한 의견
- 충돌 → 6-Lens는 정성·정량 종합, 상승여력은 정량만 → 정성 요소가 강할 때 6-Lens 우선

### Q6. AI별 답변이 너무 다른데 일관성 어떻게?
A. v9.0은 본문에 [Claude:] [ChatGPT:] [Gemini:] 점대괄호 분기로 차이를 흡수. 같은 프롬프트 + 다른 AI = 같은 골격 + 다른 도구 활용. 결과는 거의 동일.

### Q7. Quick Track과 정식 9단계의 차이?
A.
- **Quick Track**: 사업보고서 PDF 1개 + 한 명령으로 10분 (사전 스크리닝용)
- **정식 9단계**: Step 0~8 + 5.5 = 1시간 (발표용)
- Quick Track 결과 → 정식 코스의 Step 2·6 인풋으로 활용 가능

### Q8. 학회 PPT 양식과 Step 8이 안 맞아요
A. v9.0의 7페이지 표준 양식은 가이드일 뿐. 학회 양식이 5페이지면 1·7 합치고 3·4 합치는 식으로 부원 재량 조정. 핵심은 **글자 수 룰 + Mermaid + Red Team 박스 + 6-Lens 점수** 유지.

### Q9. 위클리에 직전 분석이 없으면?
A. 정식 9단계(`GIC_기업리서치_v9.md`)로 1회 풀 통과 → Step 7 결과 저장 → 이후부터 위클리.

### Q10. Top Pick과 일반 기업리서치 어떻게 구분?
A.
- **Top Pick**: 산업 → 종목 (top-down). 1~2개 발견용
- **기업리서치**: 기업 → 산업 (bottom-up). 정식 분석
- Top Pick 결과 → 발견된 종목을 정식 기업리서치로 풀 분석

2. `/gic-research 삼성전기` 호출 + yaml 붙여넣기
3. 5분 만에 학회양식.pptx + PDF 자동 생성
4. PowerPoint에서 미세 수정 → 발표

### 11.4 동등성 검증 표

| 산출물 | Gemini 시간 | ChatGPT 시간 | Claude 챗봇 시간 | Claude Code Skill 시간 |
|---|---|---|---|---|

### v10.0 작업 원칙 (v9.0 §9.1 + v10 신규 4)

#### 원칙 5 [v10 신규]
자동화는 항상 챗봇 복붙(티어 1)의 부속 옵션. v9.0 본문 한 줄도 축소하지 않음.

#### 원칙 7 [v10 신규]
