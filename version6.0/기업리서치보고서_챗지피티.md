# GIC 기업 리서치보고서 프롬프트 v6.0 — ChatGPT 최적화

> **목적**: 증권사 애널리스트 수준의 기업분석 리포트를 AI와 대화하며 단계적으로 생성한다.
> **출력물**: ① Excel(재무모델 + 밸류에이션) ② HTML(서술 분석 대시보드) ③ 양식 기반 최종 리포트
> **핵심 원칙**: 모든 숫자에는 출처를 붙이고, 모든 판단에는 근거를 제시한다.

---

## 공통 규칙

1. **언어**: 반드시 한국어로 작성한다. 영어 전문용어는 한글(영문) 형태로 병기한다.
2. **출처 필수**: 모든 숫자·순위·비율·날짜에 출처를 표기한다. 형식: `(출처: OOO, 2026.05)` 또는 각주 번호.
3. **최신 정보 우선**: 웹 검색을 통해 최신 뉴스·공시·실적을 먼저 수집하고, 검색 결과의 날짜를 명시한다.
4. **용어 설명**: 전문용어가 처음 등장할 때 괄호 안에 쉬운 설명을 넣는다.
5. **시각 자료**: 표·차트·도식이 필요한 곳에 적극적으로 제안하고 생성한다.
6. **대화형 진행**: 각 단계 완료 후 사용자에게 확인 질문을 하고, 피드백을 반영한 뒤 다음 단계로 진행한다.
7. **2트랙 출력**: 데이터 집약적 내용은 Excel/CSV로, 서술 분석은 HTML로 분리하여 출력한다.

## 모델 메모 (ChatGPT 전용 지침)

- 질문은 짧게, 근거는 분명하게, 결과는 바로 쓰기 좋게 정리한다.

- 구조를 먼저 보여주고 세부 내용을 채워나가는 방식을 우선한다.

## 6단계 — 밸류에이션 및 목표주가 산출 (Excel 출력 핵심)

### A. 미래 실적 추정 (3~5개년)

핵심 가정: 매출 성장률(사업부별 Vol×ASP), OPM, 감가상각, CAPEX, 세율, 비지배지분율

**3-Statement 모델:**
1. 손익계산서: 사업부별 매출 → 매출총이익 → 영업이익 → 순이익 → EPS
2. 재무상태표: 운전자본, 유형자산, 이익잉여금 → **Balance Check 필수**
3. 현금흐름표: 영업CF, 투자CF, 재무CF → FCF

### B. 밸류에이션 방법론

**PER 기반 (주력):**
목표주가 = NTM EPS × 적정 PER (피어 평균 / 자사 과거 / 성장 프리미엄 종합)

**PBR 기반 (금융주):**
목표주가 = Forward BPS × 적정 PBR (ROE/Ke)

**EV/EBITDA:**
목표주가 = (EBITDA × 적정 배수 - 순차입금) / 발행주식수

**DCF (선택):**
WACC, Terminal Value(g=1~2%), FCF 현재가치 합산

### C. 시나리오별 목표주가

| 시나리오 | 핵심 가정 | EPS | PER | 목표주가 | 상승여력 |
|---------|---------|-----|-----|---------|---------|
| Bear | OPM 하락, 성장 둔화 | | 피어 하단 | | |
| Base | 컨센서스 수준 | | 피어 평균 | | |
| Bull | 신규 모멘텀 | | 피어 상단 | | |

### D. 투자의견 자동 판정
- 상승여력 ≥ +15% → **BUY**
- -10% ≤ 상승여력 < +15% → **HOLD**
- 상승여력 < -10% → **SELL**
