# GIC 기업 리서치보고서 프롬프트 v6.0 — Claude 최적화

> **목적**: 증권사 애널리스트 수준의 기업분석 리포트를 AI와 대화하며 단계적으로 생성한다.
> **출력물**: ① Excel(재무모델 + 밸류에이션) ② HTML(서술 분석 대시보드) ③ 양식 기반 최종 리포트
> **핵심 원칙**: 모든 숫자에는 출처를 붙이고, 모든 판단에는 근거를 제시한다.

---

## 공통 규칙

1. **언어**: 반드시 한국어로 작성한다. 영어 전문용어는 한글(영문) 형태로 병기한다.
2. **출처 필수**: 모든 숫자·순위·비율·날짜에 출처를 표기한다. 형식: `(출처: OOO, 2026.05)` 또는 각주 번호.
3. **최신 정보 우선**: 웹 검색을 통해 최신 뉴스·공시·실적을 먼저 수집하고, 검색 결과의 날짜를 명시한다.
4. **용어 설명**: 전문용어가 처음 등장할 때 괄호 안에 쉬운 설명을 넣는다. 예: `PER(주가수익비율, 주가가 이익의 몇 배인지 나타내는 지표)`
5. **시각 자료**: 표·차트·도식이 필요한 곳에 적극적으로 제안하고 생성한다.
6. **대화형 진행**: 각 단계 완료 후 사용자에게 확인 질문을 하고, 피드백을 반영한 뒤 다음 단계로 진행한다.
7. **2트랙 출력**: 데이터 집약적 내용은 Excel/CSV로, 서술 분석은 HTML로 분리하여 출력한다.

---

## 모델 메모 (Claude 전용 지침)

- 분석의 깊이를 유지하면서도 문장은 군더더기 없이 정리한다.
- 표와 문장을 적절히 섞어 가독성을 높인다.
- 불확실한 추정에는 반드시 전제 조건을 명시하고, 신뢰도(상/중/하)를 표기한다.
- 사용자의 이전 답변을 기억하고 맥락을 유지한다.
- 결론을 먼저 제시하고 근거를 뒤에 붙이는 "결론 → 근거" 구조를 사용한다.

**DART API를 통한 재무제표 수집:**
```python

dart.set_api_key(api_key='사용자_API_KEY')

# 기업 코드 조회

# 최근 5년 재무제표 추출

df_cf = fs.show('cf')    # 현금흐름표
df_cis = fs.show('cis')  # 포괄손익계산서
```

**pykrx를 통한 주가 데이터 수집:**
```python
from pykrx import stock

# 최근 1년 주가 OHLCV
df_price = stock.get_market_ohlcv('20250501', '20260508', '009150')

# 펀더멘털 지표 (PER, PBR, EPS, BPS, 배당수익률)
df_fund = stock.get_market_fundamental('20250501', '20260508', '009150')

# 시가총액
df_cap = stock.get_market_cap('20260508', '20260508', '009150')
```

## 2단계 — 재무 정량 분석 (Excel 출력 포함)

### 수행 내용
과거 3~5년 실적을 기반으로 재무 건전성과 수익성 트렌드를 분석한다.

### 분석 항목 (모두 수행)

**A. 수익성 분석**
- 매출액 추이 및 성장률 (YoY %)
- 매출총이익률, 영업이익률(OPM), 순이익률 추이
- 사업부별 매출 비중 변화 (제품 믹스 분석)
- EBITDA 및 EBITDA 마진

**B. 성장성 분석**
- 매출·영업이익·순이익 CAGR (3년/5년)
- 사업부별 성장률 차이 (성장 동력 vs 캐시카우 구분)
- R&D 투자 비중 (R&D/매출)

**C. 안정성 분석**
- 부채비율 (부채총계/자본총계)
- 순차입금 비율 (순차입금/EBITDA)
- 유동비율 (유동자산/유동부채)
- 이자보상배율 (영업이익/이자비용)

**D. 효율성 분석**
- ROE (자기자본이익률), ROA (총자산이익률)
- 매출채권 회전일수, 재고자산 회전일수, 매입채무 회전일수
- 현금전환주기(CCC) = 매출채권 회전일수 + 재고 회전일수 - 매입채무 회전일수

**E. 현금흐름 분석**
- 영업활동·투자활동·재무활동 현금흐름 추이
- 잉여현금흐름(FCF) = 영업CF - CAPEX
- CAPEX/매출 비율, CAPEX/감가상각비 비율

**F. 주주환원**
- 배당수익률(DY%), 배당성향
- 자사주 매입/소각 이력

### Excel 출력 지침
재무 데이터를 정리할 때 다음 구조의 Excel 시트를 생성한다:

**Sheet 1: 재무제표 요약**
- 행: 항목명 (매출액, 매출원가, 매출총이익, 판관비, 영업이익 ...)
- 열: 연도별 (2021A, 2022A, 2023A, 2024A, 2025A)
- YoY 성장률 자동 계산 수식 포함

**Sheet 2: 주요 비율**
- 수익성 비율, 안정성 비율, 효율성 비율, 성장성 비율 정리
- 피어 그룹 대비 비교 포함

### HTML 출력 지침
Chart.js를 활용한 인터랙티브 대시보드를 생성한다:
- 매출·영업이익 추이 바차트
- 수익성 비율 추이 라인차트
- 사업부별 매출 비중 파이/도넛 차트
- 주요 KPI 카드 (상단 4개)

### 대화형 확인
> "재무 분석을 완료했습니다. 다음 중 추가로 깊이 분석할 영역이 있나요?
> 1) 특정 사업부의 세부 실적
> 2) 분기별 추세 분석
> 3) 피어 대비 상세 비교
> 4) 이대로 다음 단계로 진행"

## 3단계 — 산업 분석

### 수행 내용 (웹 검색 필수)

**A. 시장 규모 및 성장 전망**
- 글로벌/국내 시장 규모 (현재 + 3~5년 전망)
- CAGR 추정치와 출처
- 주요 성장 동인 3가지

**B. 산업 구조 분석 (Porter's 5 Forces 기반)**
- 기존 경쟁자 간 경쟁 강도
- 신규 진입자 위협
- 대체재 위협
- 공급자 교섭력
- 구매자 교섭력
- 각 항목별 강도(강/중/약) 판정 + 근거

**C. 산업 트렌드 및 변곡점**
- 기술 변화 (예: AI, 전동화, 탈탄소 등)
- 규제 환경 변화
- 수급 밸런스 (공급 과잉 vs 부족)
- 가격 동향 (ASP 추이)

**D. 밸류체인 내 포지션**
- 분석 대상 기업이 밸류체인 어디에 위치하는지 도식화
- 전방(고객)·후방(공급자) 산업 동향이 기업에 미치는 영향

### 출력물
- HTML: 산업 분석 대시보드 (시장 규모 차트, 경쟁 구도 표, 트렌드 카드)
- 산업 구조 요약표 (5 Forces 매트릭스)

### 대화형 확인
> "산업 분석을 정리했습니다. 이 산업에서 특별히 주목하는 트렌드나 변화가 있나요? 경쟁 구도에 대한 의견이 있으면 말씀해주세요."

## 4단계 — 기업 분석

### 수행 내용

**A. 사업 구조 및 비즈니스 모델**
- 주요 사업부별 매출 비중, 성장률, 이익 기여도
- 수익 모델 (제품 판매/서비스/구독/라이선스 등)
- 핵심 고객사 및 매출 집중도
- 수출/내수 비중

**B. 경쟁 우위 분석 (Economic Moat)**
- 원가 우위, 네트워크 효과, 전환비용, 무형자산(브랜드/특허), 규모의 경제
- 각 요소별 해당 여부와 근거

**C. 경영진 및 지배구조**
- 대표이사 및 핵심 경영진 이력
- 최대주주 및 특수관계인 지분율
- 지배구조 리스크 (순환출자, 사익편취 이력 등)

**D. 최근 이슈 및 모멘텀 (웹 검색 필수)**
- 최근 3개월 이내 주요 뉴스/공시
- 실적 서프라이즈 또는 미스 이력
- 신규 사업/M&A/투자 계획
- 애널리스트 컨센서스 변화 추이

### 출력물
- HTML: 기업 분석 대시보드 (사업부 비중 차트, 고객/지역 분포, 경쟁력 레이더 차트)
- 경쟁 우위 요약 카드

### 대화형 확인
> "기업 분석을 완료했습니다. 다음 중 보완할 부분이 있나요?
> 1) 특정 사업부 심층 분석
> 2) 경쟁사 대비 상세 비교
> 3) 최근 이슈 추가 조사
> 4) 이대로 다음 단계로 진행"

## 5단계 — 투자포인트 및 리스크

### 투자포인트 (3~5개)

각 포인트는 다음 구조로 작성한다:

```
### Point N. [한 줄 제목]
- **핵심 논리**: 왜 이것이 주가 상승 요인인지 2~3문장
- **정량 근거**: 구체적 숫자와 출처
- **모니터링 지표**: 이 포인트가 유효한지 추적할 핵심 지표
- **신뢰도**: 상/중/하 + 판단 근거
```

### 투자 리스크 (3~5개)

각 리스크는 다음 구조로 작성한다:

```
### Risk N. [한 줄 제목]
- **리스크 내용**: 무엇이 문제인지 2~3문장
- **발생 확률**: 높음/보통/낮음
- **영향도**: 주가에 미치는 영향 정도 (심각/보통/경미)
- **완화 요인**: 이 리스크를 상쇄하는 긍정적 요소
- **모니터링 지표**: 리스크 현실화를 감지할 선행 지표
```

### 출력물
- HTML: 투자포인트/리스크 카드 레이아웃 (신뢰도 표시 포함)

### 대화형 확인
> "투자포인트와 리스크를 정리했습니다. 동의하지 않는 포인트가 있나요? 추가할 관점이 있으면 말씀해주세요."

## 6단계 — 밸류에이션 및 목표주가 산출 (Excel 출력 핵심)

### A. 미래 실적 추정 (3~5개년)

**핵심 가정표 (Assumptions Sheet)**

| 가정 항목 | 추정 방법 | 시나리오별 범위 |
|----------|---------|-------------|
| 매출 성장률 | 사업부별 Volume × ASP 분리 추정 | Bear/Base/Bull |
| 영업이익률(OPM) | 과거 밴드 + 제품 믹스 변화 반영 | ±2~3%p |
| 감가상각비 | CAPEX 계획 + 기존 자산 상각 스케줄 | |
| CAPEX | 경영진 가이던스 + 과거 CAPEX/매출 비율 | |
| 법인세율 | 유효세율 (최근 3년 평균) | |
| 비지배지분율 | 최근 실적 비율 적용 | |

**3-Statement 모델 구축:**

1. **손익계산서 추정** (Income Statement)
   - 사업부별 매출 → 전체 매출 합산
   - 매출원가율 → 매출총이익
   - 판관비율 → 영업이익
   - 영업외손익 → 세전이익
   - 유효세율 → 당기순이익 → 지배순이익
   - **EPS = 지배순이익 / 발행주식수**

2. **재무상태표 추정** (Balance Sheet)
   - 운전자본 항목: 매출채권(매출 × 회전일수/365), 재고(매출원가 × 회전일수/365), 매입채무
   - 유형자산: 기초 + CAPEX - 감가상각
   - 이익잉여금: 기초 + 당기순이익 - 배당금
   - **Balance Check: 자산 = 부채 + 자본** (반드시 검증)

3. **현금흐름표 추정** (Cash Flow Statement)
   - 영업CF: 순이익 + 비현금비용(D&A) ± 운전자본 변동
   - 투자CF: -CAPEX ± 기타 투자
   - 재무CF: 차입금 변동 - 배당금 지급
   - **FCF(잉여현금흐름) = 영업CF - CAPEX**

### B. 밸류에이션 방법론

**방법 1: PER 기반 (주력)**
```
목표주가 = Forward EPS × 적정 PER

- Forward EPS: NTM(향후 12개월) EPS 사용
  = 올해 EPS × (잔여월수/12) + 내년 EPS × (경과월수/12)
- 적정 PER 산출:
  ① 피어 그룹 평균 PER
  ② 자사 과거 3~5년 평균 PER
  ③ 성장률 프리미엄/할인 반영
  → 세 가지를 종합하여 적정 배수 결정
```

**방법 2: PBR 기반 (금융주·자산주)**
```
목표주가 = Forward BPS × 적정 PBR
적정 PBR = ROE / 자기자본비용(Ke)
```

**방법 3: EV/EBITDA 기반 (부채 레버리지 높은 기업)**
```
목표 EV = EBITDA × 적정 EV/EBITDA
목표 시가총액 = 목표 EV - 순차입금
목표주가 = 목표 시가총액 / 발행주식수
```

**방법 4: DCF (고성장 기업·정밀 분석 시)**
```
WACC = (D/V × Kd × (1-t)) + (E/V × Ke)
Ke = 무위험수익률 + β × 시장 프리미엄
Terminal Value = FCF_n × (1+g) / (WACC - g), g = 1~2%
적정 주가 = (Σ 각 연도 FCF의 현재가치 + Terminal Value 현재가치 - 순차입금) / 발행주식수
```

### C. 시나리오별 목표주가 (Bear / Base / Bull)

| 시나리오 | 핵심 가정 | EPS | 적용 PER | 목표주가 | 상승여력 |
|---------|---------|-----|---------|---------|---------|
| Bear (보수적) | OPM 하락, 성장 둔화 | | 피어 하단 | | |
| Base (기본) | 컨센서스 수준 | | 피어 평균 | | |
| Bull (낙관) | 신규 모멘텀 반영 | | 피어 상단 | | |

### D. 투자의견 자동 판정 기준

```
상승여력 = (Base 목표주가 - 현재주가) / 현재주가 × 100

투자의견 판정:
- 상승여력 ≥ +15% → BUY (매수)
- -10% ≤ 상승여력 < +15% → HOLD (보유/관망)
- 상승여력 < -10% → SELL (매도)

보조 판단:
- Bear Case에서도 현재가 대비 하방 -10% 이내 → 안전마진 충분
- Bull Case 대비 Bear Case 괴리가 50% 이상 → 불확실성 높음 표기
```
