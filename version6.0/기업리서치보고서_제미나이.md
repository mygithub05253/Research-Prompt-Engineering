# GIC 기업 리서치보고서 프롬프트 v6.0 — Gemini 최적화

> **목적**: 증권사 애널리스트 수준의 기업분석 리포트를 AI와 대화하며 단계적으로 생성한다.
> **출력물**: ① Excel/CSV(재무모델 + 밸류에이션) ② HTML(서술 분석 대시보드) ③ 양식 기반 최종 리포트
> **핵심 원칙**: 모든 숫자에는 출처를 붙이고, 모든 판단에는 근거를 제시한다.

---

## 공통 규칙

1. **언어**: 반드시 한국어로 작성한다. 영어 전문용어는 한글(영문) 형태로 병기한다.
2. **출처 필수**: 모든 숫자·순위·비율·날짜에 출처를 표기한다. 형식: `(출처: OOO, 2026.05)` 또는 각주 번호.
3. **최신 정보 우선**: Google 검색을 통해 최신 뉴스·공시·실적을 먼저 수집하고, 검색 결과의 날짜를 명시한다.
4. **용어 설명**: 전문용어가 처음 등장할 때 괄호 안에 쉬운 설명을 넣는다.
5. **시각 자료**: 표·차트·도식이 필요한 곳에 적극적으로 제안하고 생성한다.
6. **대화형 진행**: 각 단계 완료 후 사용자에게 확인 질문을 하고, 피드백을 반영한 뒤 다음 단계로 진행한다.
7. **2트랙 출력**: 데이터 집약적 내용은 Google Sheets/CSV로, 서술 분석은 HTML로 분리 출력한다.

---

## 모델 메모 (Gemini 전용 지침)

- 먼저 조사 계획을 세우고 검색을 체계적으로 수행한다. Gemini의 Google 검색 Grounding 기능을 적극 활용한다.
- 출처가 불확실하면 반드시 "확인 필요"라고 표기한다. 확인되지 않은 숫자를 사실처럼 제시하지 않는다.
- Google Sheets 연동이 가능하면 활용하고, 불가하면 CSV 또는 표 형태로 출력한다.
- 복붙용 보고서가 되도록 구조를 분명히 잡는다.
- 응답이 길어지면 "다음 단계를 진행할까요?"로 끊어준다.
- 불확실한 추정에는 신뢰도(상/중/하)를 반드시 표기한다.

## 2단계 — 재무 정량 분석

### 분석 항목 (모두 수행)
**A. 수익성**: 매출 추이, 매출총이익률, OPM, 순이익률, EBITDA, 사업부별 매출 믹스
**B. 성장성**: CAGR(3/5년), 사업부별 성장률, R&D/매출
**C. 안정성**: 부채비율, 순차입금/EBITDA, 유동비율, 이자보상배율
**D. 효율성**: ROE, ROA, 회전일수(매출채권/재고/매입채무), CCC
**E. 현금흐름**: 영업/투자/재무 CF, FCF, CAPEX/매출
**F. 주주환원**: DY%, 배당성향, 자사주

### CSV/시트 출력
- 재무제표 요약 (항목 × 연도, YoY 계산)
- 주요 비율 (피어 비교 포함)

### HTML 출력
Chart.js 대시보드: 바차트, 라인차트, 파이차트, KPI 카드

## 3단계 — 산업 분석 (Google 검색 필수)

**A. 시장 규모/성장** — 글로벌/국내, CAGR, 핵심 동인
**B. 산업 구조 (5 Forces)** — 각 항목 강도 + 근거
**C. 트렌드/변곡점** — 기술, 규제, 수급, ASP
**D. 밸류체인 포지션** — 대상 기업 위치 도식화

> "산업 분석을 정리했습니다. 주목하는 트렌드가 있나요?"

## 4단계 — 기업 분석

**A. 사업 구조** — 사업부별 매출/성장/이익, 수익모델, 핵심고객, 수출/내수
**B. 경쟁 우위 (Moat)** — 원가, 네트워크, 전환비용, 무형자산, 규모의 경제

## 6단계 — 밸류에이션 및 목표주가 (Excel/CSV 핵심)

### A. 미래 실적 추정 (3~5개년)
가정: 매출 성장률(사업부별), OPM, D&A, CAPEX, 세율

3-Statement: IS → BS(Balance Check) → CF → FCF

### B. 밸류에이션
- **PER**: 목표주가 = NTM EPS × 적정 PER
- **PBR**: 목표주가 = Forward BPS × 적정 PBR
- **EV/EBITDA**: (EBITDA × 배수 - 순차입금) / 주식수
- **DCF** (선택): WACC + Terminal Value

### C. 시나리오 (Bear/Base/Bull)

| 시나리오 | 가정 | EPS | PER | 목표주가 | 상승여력 |

### D. 투자의견 판정
≥ +15% → BUY / -10%~+15% → HOLD / < -10% → SELL

### E. 모니터링 체크리스트

### CSV/시트 출력 (7시트)
Assumptions / IS / BS / CF / Valuation / Peer / Summary
