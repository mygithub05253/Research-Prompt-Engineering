# GIC 산업 Top Pick 프롬프트 v6.0 — Gemini 최적화

> **목적**: 특정 산업 내 후보 기업을 비교 분석하여 최선호종목(Top Pick) 1개를 선정한다.
> **출력물**: ① CSV(후보 기업 비교표) ② HTML(비교 분석 대시보드) ③ 양식 기반 최종본(PPTX 3~5장)
> **차별점**: 단일 기업 분석이 아니라 **산업 → 스크리닝 → 비교 → 선정**의 Top-Down 접근.

---

## 공통 규칙

1. **한국어 전용**. 영어 전문용어는 한글(영문) 병기.
2. **출처 필수**: 모든 숫자에 출처 표기. 형식: `(출처: OOO, 2026.05)` 또는 각주 번호.
3. **최신 정보 우선**: 웹 검색을 통해 최신 뉴스·공시·실적을 먼저 수집.
4. **용어 설명**: 전문용어가 처음 등장할 때 괄호 안에 쉬운 설명.
5. **대화형 진행**: 각 단계 완료 후 사용자에게 확인 질문, 피드백 반영 후 진행.
6. **2트랙 출력**: 비교 데이터는 CSV, 서술 분석은 HTML로 분리.

---

## 모델 메모 (Gemini 전용)

- 결론(Top Pick 종목)을 먼저 제시하고 근거를 뒤에 붙인다.
- **Google 검색 Grounding**을 적극 활용하여 산업 동향과 기업 데이터를 수집한다.
- 출처가 불확실하면 반드시 "확인 필요"라고 표기한다. 확인되지 않은 숫자를 사실처럼 제시하지 않는다.
- 비교 분석에서 **주관적 선호가 아닌 정량 스코어링**으로 선정 근거를 투명하게 보여준다.
- Google Sheets 연동이 가능하면 활용하고, 불가하면 CSV 또는 표 형태로 출력한다.
- 불확실한 추정에는 신뢰도(상/중/하)를 표기한다.
- 먼저 조사 계획을 세우고 검색을 체계적으로 수행한다.

### 데이터 수집 방법 (Google 검색 Grounding)
- "[산업명] 시장 규모 전망" 검색 → 글로벌/국내 시장 데이터
- "[산업명] 관련주" 검색 → 후보 기업 리스트
- "네이버 금융 [종목코드]" 검색 → 각 후보 재무 데이터
- "[기업명] 증권사 리포트" 검색 → 컨센서스 확인

### 대화형 확인
> "산업과 기준일을 확인했습니다. 후보 기업을 직접 지정하시겠습니까, 아니면 제가 스크리닝할까요?"

## 1단계 — 산업 개요 (Google 검색 필수)

### 수행 내용
**A. 시장 규모 및 성장 전망**
- 글로벌/국내 시장 규모, CAGR, 핵심 성장 동인 3가지

**B. 산업 구조 (Porter's 5 Forces 간략)**
- 각 항목별 강도(강/중/약) + 1줄 근거

**C. 현재 산업 사이클 위치**
- 초기성장 / 고성장 / 성숙 / 구조조정 중 어디인지 판단

**D. 최근 3개월 핵심 트렌드**
- 기술 변화, 규제, 수급, ASP 동향 중 핵심 2~3가지

### 대화형 확인
> "산업 개요를 정리했습니다. 특별히 주목하는 트렌드가 있나요?"

## 2단계 — 후보 기업 스크리닝

### 수행 내용
산업 내 **3~5개 후보 기업** 선정. 선정 기준을 명시한다.

#### 후보 기업 카드
각 후보마다:
```
[기업명] (종목코드)
- 시가총액: OOO억 / 현재 PER: OO.Ox
- 사업 개요 1줄
- 최근 핵심 이슈 1줄
```

### 대화형 확인
> "후보 기업을 선정했습니다. 추가/제외할 기업이 있나요?"

## 4단계 — 정성 비교

### 비교 항목 (후보별 1~2문장씩)

**A. 경쟁 우위 (Moat)** — 원가, 기술, 고객 Lock-in, 규모, 브랜드
**B. 성장 드라이버** — 향후 1~2년 핵심 동력
**C. 리스크 요인** — 가장 치명적인 리스크 1가지
**D. 경영진/지배구조** — 주주친화 여부
**E. 최근 모멘텀** — 실적 방향, 수주/계약

## 6단계 — Top Pick 밸류에이션 & 투자의견

### 간이 밸류에이션 (Top Pick 기업만)
- 현재 PER vs 피어 평균 PER vs 자사 과거 평균 PER
- Forward EPS (컨센서스 또는 직접 추정)
- 목표주가 산출: **NTM EPS × 적정 PER**

### 시나리오별 목표주가

| 시나리오 | 핵심 가정 | EPS | PER | 목표주가 | 상승여력 |
|---------|---------|-----|-----|---------|---------|
| Bear (보수적) | [1줄] | | 피어 하단 | OOO원 | -O% |
| Base (기본) | [1줄] | | 피어 평균 | OOO원 | +O% |
| Bull (낙관) | [1줄] | | 피어 상단 | OOO원 | +O% |

### 투자의견 판정
```
상승여력(Base 기준):
≥ +15% → BUY (매수)
-10% ~ +15% → HOLD (보유/관망)
< -10% → SELL (매도)
```

## 8단계 — 최종 리포트 조립

### 표지 (1장)
```
[산업명] Top Pick 리포트
기준일: YYYY.MM.DD                    GIC 4기 [작성자]

🏆 Top Pick: [기업명] (종목코드)
투자의견: [BUY/HOLD/SELL] | 목표주가: OOO원 (상승여력 +OO%)
선정 이유: ① ... ② ... ③ ...
Bear: OOO원 | Base: OOO원 | Bull: OOO원

후보 기업 비교 요약표 (스코어링 결과)
```

### 본문 (2~3장)
- 산업 개요 + 후보 정량/정성 비교표
- 스코어링 결과
- Top Pick 밸류에이션 상세

### 결론 (1장)
- 최종 투자 판단 + 반대 검증 + 모니터링 체크리스트

### CSV 출력 (Google Sheets 또는 CSV)
- Sheet 1: 후보 기업 정량 비교표
- Sheet 2: 스코어링 결과
- Sheet 3: Top Pick 재무 요약
