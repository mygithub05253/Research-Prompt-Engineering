# GIC 위클리 투자리포트 프롬프트 v6.0 — ChatGPT 최적화

> **목적**: 이번 주 핵심 변화를 짚어 빠르게 투자 판단을 돕는 주간 리포트를 생성한다.
> **출력물**: ① CSV(핵심 지표 요약) ② HTML(1페이지 리포트) ③ 양식 기반 최종본(PPTX 3장)
> **차별점**: 기업리서치보고서 대비 간결·핵심 중심. 분석 깊이보다 시의성과 판단의 명확성이 핵심.

---

## 공통 규칙

1. **한국어 전용**. 영어 전문용어는 한글(영문) 병기.
2. **출처 필수**: 모든 숫자에 출처 표기.
3. **최신 정보 우선**: 이번 주(최근 7일) 뉴스·공시·실적을 웹 검색으로 수집.
4. **용어 설명**: 어려운 용어는 처음 등장 시 쉬운 설명 병기.
5. **대화형**: 초기 설정 후 최소한의 확인만 거치고 빠르게 진행.
6. **간결함 우선**: 전체 리포트가 PPTX 3장(표지+본문+결론) 이내로 완결.

---

## 모델 메모 (ChatGPT 전용)

- 짧은 보고서처럼 깔끔하게 마무리한다. 1문장이면 충분한 것을 3문장으로 늘리지 않는다.
- 결론(BUY/HOLD/SELL)을 먼저 제시하고 근거를 뒤에 붙인다.
- **Advanced Data Analysis(코드 인터프리터)**를 활용하여 CSV 파일을 직접 생성한다.
- Python pandas로 재무 데이터를 정리하고, matplotlib(한글 폰트: NanumGothic)으로 차트를 생성한다.
- 불확실한 추정에는 신뢰도를 표기하되, 위클리에서는 "확신/보통/불확실" 3단계로 간소화.
- Bear/Base/Bull 시나리오를 반드시 포함한다.
- 구조를 먼저 보여주고 세부 내용을 채워나가는 방식을 우선한다.

## 0단계 — 초기 설정

### 확인 사항 (필수만, 빠르게)
| 항목 | 필수 |
|------|------|
| 기업명 + 종목코드 | 필수 |

| 독자 수준 | 선택(기본: 중급) |

### 자동 수집 데이터
- 현재 주가, 주간 등락률, 거래량 변화
- 이번 주 주요 뉴스/공시 (웹 검색)
- 최근 분기 실적 (DART/네이버 금융)
- 증권사 목표주가 컨센서스 변화

### 데이터 수집 코드 (Advanced Data Analysis 활용 시)
```python
# pykrx를 통한 주가 데이터 수집
from pykrx import stock
import pandas as pd

# 주간 주가 데이터
df_price = stock.get_market_ohlcv('20260501', '20260508', '종목코드')
df_fund = stock.get_market_fundamental('20260508', '20260508', '종목코드')
df_cap = stock.get_market_cap('20260508', '20260508', '종목코드')

# 주간 등락률 계산
weekly_return = (df_price['종가'].iloc[-1] / df_price['종가'].iloc[0] - 1) * 100
```

**API 미사용 시:** 웹 검색으로 네이버 금융, DART 공시 데이터를 수집한다.

## 1단계 — 이번 주 핵심 변화 3가지

### 수행 내용
이번 주에 해당 기업/산업에서 일어난 가장 중요한 변화 **3개만** 선별한다.

각 변화는 다음 구조:
```
### 변화 N. [한 줄 제목]
- **무슨 일이 있었나**: 팩트 1~2문장
- **왜 중요한가**: 주가/실적에 미치는 영향 1문장
- **출처**: (OOO, YYYY.MM.DD)
```

---

## 2단계 — 핵심 투자포인트 3개

기업리서치보고서의 투자포인트에서 **이번 주 관점에서 가장 중요한 3개**만 추출.

각 포인트:
```
Point N. [한 줄 제목]
- 핵심 논리 1~2문장
- 정량 근거 (숫자 + 출처)
- 확신도: 확신/보통/불확실
```

---

## 3단계 — 리스크 1~2개

과하게 무겁지 않게, 꼭 알아야 할 리스크만 간결하게.

```
Risk N. [한 줄 제목]
- 내용 1~2문장
- 영향도: 심각/보통/경미
```

## 4단계 — 밸류에이션 & 투자의견

### 간이 밸류에이션
- 현재 PER vs 피어 평균 PER vs 자사 과거 평균 PER
- Forward EPS (최근 컨센서스 또는 직접 추정)
- 목표주가 산출: **NTM EPS × 적정 PER**

### 시나리오별 목표주가

| 시나리오 | 핵심 가정 | 목표주가 | 상승여력 |
|---------|---------|---------|---------|
| Bear (보수적) | [1줄 설명] | OOO원 | -O% |
| Base (기본) | [1줄 설명] | OOO원 | +O% |
| Bull (낙관) | [1줄 설명] | OOO원 | +O% |

### 투자의견 판정
```
상승여력(Base 기준):
≥ +15% → BUY (매수)
-10% ~ +15% → HOLD (보유/관망)
< -10% → SELL (매도)
```

### 단기(3~6개월) vs 중기(1년) 의견 분리
- **단기**: 모멘텀·이벤트 기반 판단
- **중기**: 밸류에이션 기반 판단
- 둘이 다를 수 있음 (예: 단기 HOLD, 중기 BUY = "분할 매수 구간")

## 6단계 — 개인 의견 (분석자 주관)

### 증권사 리포트와의 비교
