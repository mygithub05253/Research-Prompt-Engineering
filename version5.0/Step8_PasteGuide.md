# Step 8 (v5.0) — 복붙 가이드 (Paste-Ready Assembly Guide)

> **패러다임 전환**: 파일 생성 ❌ → 부원이 동아리 PPT 양식에 그대로 복사-붙여넣기 ✅

---

## 프롬프트 본문

```
[Step 8 — 최종 리포트 복붙 가이드 생성]

지금부터 최종 리서치 보고서를 작성합니다.

[중요 주의사항 — 절대 위반 금지]
1. PDF, PPTX, DOCX 파일을 직접 생성하지 마라.
2. 다운로드 링크를 만들지 마라.
3. Python·코드 실행으로 파일을 만들지 마라.
4. "파일을 첨부했습니다" 같은 거짓 진술 금지.

대신 부원이 동아리 공식 PPT 양식에 슬라이드별로 '복사-붙여넣기'만
하면 되도록, 슬라이드 번호 → 텍스트 박스 위치 → 본문 순으로
완벽하게 구조화된 텍스트만 출력하라.

──────────────────────────────────────────
[출력 양식 — 1페이지부터 7페이지까지 모두 채울 것]

[1페이지: 커버 슬라이드]
- 메인 타이틀 (≤25자):
  → [기업명] - [핵심 투자포인트 1줄 요약]
- 서브 타이틀 (≤40자):
  → [업종/섹터] | 분석 기준일 [YYYY.MM.DD]
- 투자의견 박스:
  → 투자의견 [BUY/HOLD/SELL] | 목표주가 [###,###원] | 상승여력 [##.#%]
- 작성자 / 동아리명:
  → GIC | [작성자명]

[2페이지: Executive Summary]
- 좌측 본문 (≤200자, 줄바꿈 자유):
  → [3줄 요약: 회사가 무엇을 하는지 / 왜 지금 사야 하는지 / 무엇을 조심해야 하는지]
- 우측 핵심 지표 박스 (4개):
  → 매출 [###억] | 영업이익 [###억] | 영업이익률 [##.#%] | ROE [##.#%]
- 하단 표 삽입용 데이터 (3개년 마크다운 표):
  | 항목 | 2023 | 2024 | 2025E |
  |---|---|---|---|
  | 매출액 | | | |
  | 영업이익 | | | |
  | 순이익 | | | |
  | 영업이익률 | | | |

[3페이지: 산업 분석]
- 좌측 상단 본문 (≤150자):
  → [산업 핵심 트렌드 1~2개 + 회사 포지션]
- 좌측 하단 핵심 비유 (≤30자):
  → "이 회사는 [산업]의 [비유 대상] 같은 존재"
- 우측 상단 그래프 들어갈 자리 안내:
  → (여기에 '글로벌 시장 규모 추이 그래프' 캡처본을 삽입하세요)
- 우측 하단 Mermaid 다이어그램:
  ```mermaid
  flowchart LR
    A[원자재] --> B[부품] --> C[조립]
    C --> D[유통] --> E[고객]
  ```
  → (Mermaid 코드는 mermaid.live에서 PNG로 변환 후 삽입)

[4페이지: 기업 분석]
- 좌측 본문 (≤200자):
  → [지배구조 / 주요 사업부 / 매출 구성]
- 우측 사업부 매출 비중 표 (마크다운):
  | 사업부 | 매출 비중 | YoY 성장률 |
  |---|---|---|
- 하단 Mermaid 지배구조도:
  ```mermaid
  flowchart TD
    P[지주사] --> A[사업부 A]
    P --> B[사업부 B]
  ```

[5페이지: 투자포인트 & 리스크]
- 투자포인트 (3개, 각 ≤80자):
  1) [포인트 1: 매개변수 명시 — 점유율↑/마진↑/멀티플 리레이팅 등]
  2) [포인트 2]
  3) [포인트 3]
- Red Team 검증 결과 1줄 요약 (≤60자):
  → [방어 가능한 핵심 논리 / 살아남은 포인트만]
- 리스크 (2개, 각 ≤60자):
  1) [리스크 1]
  2) [리스크 2]

[6페이지: 밸류에이션]
- 적용 멀티플 박스:
  → PER [##.#x] / EV/EBITDA [##.#x] / DCF 목표가 [###,###원]
- 목표주가 산출 표 (마크다운):
  | 방법론 | 적용 배수/할인율 | 산출 가격 | 가중치 |
  |---|---|---|---|
- 우측 본문 (≤180자):
  → [멀티플 선정 근거 + Peer 비교]
- 그래프 자리 안내:
  → (여기에 'Peer Group 멀티플 비교 차트' 캡처본 삽입)

[7페이지: 결론 & Disclaimer]
- 결론 본문 (≤180자):
  → [투자의견 재확인 + 핵심 트리거 이벤트 + 모니터링 지표]
- Disclaimer 박스 (고정 문구):
  → "본 보고서는 GIC 동아리 학습 목적으로 작성되었으며,
     투자 권유가 아닙니다. 투자 결정은 본인 책임입니다."

- 초과 시 핵심 키워드 위주로 압축 후 재출력.

- 노드 5~9개 이내, 한국어 노드명 허용.

[자체 점검 체크리스트 — 출력 직전 확인]

□ 글자 수 제한을 모두 지켰는가?
□ 마크다운 표는 컬럼 정렬이 깨지지 않는가?
□ Mermaid 코드는 ```mermaid 으로 감쌌는가?

## 이 Step의 설계 의도

| 문제 (v4.0) | 해결 (v5.0) |
|---|---|
| AI가 PPT 디자인 템플릿을 못 보니 양식이 깨짐 | AI에게 디자인은 맡기지 않고, **텍스트만** 만들게 함 |
| 글자 수 통제 불가 → 텍스트 박스 삐져나감 | (≤NN자) 제한을 프롬프트 단계에서 강제 |
| 비전공자가 결과물을 어떻게 옮길지 막막 | 슬라이드 번호 → 박스 위치 → 본문 순서로 명세 |
| 그래프·이미지를 AI가 어색하게 시도 | "여기에 캡처본 삽입" 안내 + Mermaid 코드만 제공 |
| 디자인 템플릿이 흔들리고 폰트·색상이 깨짐 | 동아리 공식 양식에 그대로 복붙 → 템플릿 100% 보존 |

## 참고 — 비전공자 부원 사용 흐름

1. Step 7까지 챗봇에서 진행 → 분석 데이터 누적.
2. Step 8 프롬프트 복붙 → 챗봇이 위 양식대로 출력.
3. 동아리 PPT 양식 파일 열기.
4. 1페이지부터 차례대로 텍스트 박스에 복사-붙여넣기.
5. Mermaid 코드는 [mermaid.live](https://mermaid.live)에 붙여넣고 PNG 다운로드 → 슬라이드에 이미지로 삽입.
6. "그래프 자리" 안내된 곳에는 Step 2에서 받은 차트 캡처를 삽입.
