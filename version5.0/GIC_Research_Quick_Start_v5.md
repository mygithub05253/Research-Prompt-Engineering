# GIC 리서치 프롬프트 v5.0 — Quick Start

> 30초 안에 시작하는 가이드. 자세한 내용은 `GIC_Research_Prompt_System_v5.md` 참조.

---

## v5.0이 v4.0과 다른 점 (한 눈에)

1. **외부 도구 불필요** — ChatGPT나 Claude 챗봇 창에서 100% 완결됩니다.
2. **AI는 PPT를 만들지 않습니다** — 부원이 동아리 양식에 복붙할 텍스트만 출력합니다.
3. **모르는 단어는 자동 번역** — `ROE(자기자본 대비 이익률)` 식으로 풀이가 붙어요.
4. **Short Seller 검증 단계 추가** — 발표 전에 가상의 공매도 전문가가 약점을 잡아 줍니다.

---

## 9단계 흐름

```
Step 0  →  초기 설정 (기업·섹터·관점 정하기)
Step 1  →  경쟁사 매핑 (Peer Group)
Step 2  →  재무 분석          📎 DART 재무제표 첨부
Step 3  →  산업 분석           🪄 핵심 비유 + Mermaid
Step 4  →  기업 분석           🪄 Mermaid 지배구조
Step 5  →  투자포인트 & 리스크 🧪 Sanity Check
Step 5.5 → Red Team           ⚔️ Short Seller 공격 → 방어
Step 6  →  밸류에이션          (목표주가·투자의견)
Step 7  →  검토 & 수정         (체크리스트)
Step 8  →  복붙 가이드          📋 1~7페이지 텍스트만 출력
```

---

## 두 종류 부원을 위한 팁

**금융 지식 있고 프롬프트 처음**
- Step 0~8 순서대로 그냥 복붙하세요. 각 단계가 챗봇에게 시킬 말을 다 적어 두었습니다.
- 답변이 길면 "위 내용을 사이드바 키워드 2줄로 압축해 줘"처럼 후속 요청만 하세요.

**프롬프트 익숙하지만 금융이 어렵다**
- 모르는 용어는 답변 안에서 자동으로 풀이가 붙습니다 (블록 A).
- 산업 분석이 막히면 Step 3 첫 줄 "이 회사는 OO의 △△ 같은 존재" 비유부터 보세요.
- 투자포인트가 너무 추상적이면 Step 5.5 Red Team을 돌려서 약점을 먼저 노출시키세요.

## Step 8 — 가장 중요한 변경

### 옛날 방식 (v4.0)
> "AI야, PPT 파일 만들어 줘" → 파일이 나오긴 하는데 양식이 깨지고 글자가 박스 밖으로 삐져나감.

### 새 방식 (v5.0)
> "AI야, 1페이지 메인 타이틀 25자, 2페이지 본문 200자… 이렇게 텍스트만 줘"
> → 부원이 동아리 공식 PPT 양식 열고 그대로 복붙. 디자인 100% 보존.

### 사용법
1. Step 8 프롬프트를 챗봇에 붙여넣기.
2. 챗봇이 1페이지부터 7페이지까지 슬라이드별 텍스트를 줍니다.
3. 동아리 PPT 양식 파일 → 텍스트 박스 클릭 → 붙여넣기.
4. Mermaid 코드는 https://mermaid.live 에 붙여넣어 PNG 다운로드 → 슬라이드 이미지로 삽입.
5. "(여기에 그래프 캡처본 삽입)" 안내된 자리에는 Step 2에서 만든 차트 캡처를 직접 붙여넣기.

## 자주 막히는 지점 & 해결

| 증상 | 원인 | 해결 |
|---|---|---|
| 답변에 영어 약어가 너무 많아 이해 안 됨 | 블록 A가 호출 안 됨 | "블록 A를 적용해서 다시 출력해 줘"라고 요청 |
| 투자포인트가 "그냥 좋다" 수준 | Sanity Check 미실행 | "블록 B로 1차원 인과를 보강해 줘"라고 요청 |
| 발표 때 무슨 질문이 나올지 모르겠다 | Step 5.5 누락 | Step 5.5 Red Team 단독 실행 |
| PPT 텍스트 박스에 글자가 안 들어감 | 글자 수 초과 | "Step 8의 ≤NN자 제한을 다시 적용해서 압축" |
| 마크다운 표가 PPT에서 깨짐 | 표를 그대로 붙여넣음 | PPT의 [표 삽입] 기능 사용 후 셀에 값만 복사 |
