## 위클리 사용 워크플로

### 직전 분석이 없는 경우
1. 먼저 `GIC_기업리서치_v9.md`로 정식 9단계 풀 통과
2. Step 7 결과(투자의견·목표주가·6-Lens 점수) 저장
3. 이후부터 매주 위클리 사용

### 직전 분석이 있는 경우
1. 위 프롬프트 복사 + ■ 항목들 채우기
2. AI 챗봇에 붙여넣기
3. 답변 검토 → 4페이지 PPT 양식에 슬라이드별 복붙
4. 다음 주 위클리 시 이 답변을 "직전 위클리"로 사용

### 이번 주 새 정보가 거의 없을 때
"이번 주 정보 변동 없음. 직전 의견 유지" 1줄로 마무리. 4페이지 강제 채울 필요 없음.

## AI별 위클리 활용 팁 (본문 [점대괄호] 분기와 동일)

### Claude
- Artifacts로 직전 → 이번 주 변동 비교 차트 생성
- PDF 첨부로 분기 실적 발표 자료 직접 분석
- 회의주의 강점 — 컨센서스 변화에 자체 판단 추가

### ChatGPT
- 웹 브라우징으로 최근 7일 뉴스 자동 수집 + URL 첨부
- Code Interpreter로 변동 표 자동 계산
- GPTs Memory로 직전 위클리 자동 기억 (Plus)

### Gemini
- Search Grounding으로 출처 URL 자동 첨부 (가장 강력)
- 유튜브 분석 — 실적 발표·IR Day 영상 자동 트랜스크립트 (Advanced)
- Google Sheets 연동으로 누적 위클리 자동 기록 (Advanced)
