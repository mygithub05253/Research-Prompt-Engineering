# 산업 Top Pick — ChatGPT 최적화 (v8.0)

> 본문은 `산업_최선호종목_클로드.md`와 동일.
> ChatGPT 차이점만 아래 명시.

---

## ChatGPT 추가 지침

```
[Step 0 ChatGPT 추가 — Top Pick 전용]

1. **웹 브라우징 풀가동**
   - Phase 1 산업 진단: 최신 산업 리포트 5개 자동 검색·요약
   - Phase 2 스크리닝: 후보 5~10개 종목의 최신 멀티플 자동 수집
   - Phase 4 Top Pick: 최근 3개월 뉴스 5건 자동 수집

2. **Code Interpreter 활용 (Phase 2)**
   - 후보 종목 5~10개의 데이터를 Python으로 정렬
   - 점수 계산 자동화
   - 결과를 Excel 파일로 다운로드

3. **DALL-E (선택)**
   - 산업 비유를 일러스트로 (1p 커버 이미지)
   - 단, 발표용 슬라이드에 사용 시 학회 양식과 어울리는지 확인

4. **GPTs 활용**
   - "GIC Industry TopPick GPT" 만들어 Phase 1~6 자동 트리거
   - 매주 같은 산업 추적 시 일관성 유지
```

---

## ChatGPT Top Pick 강점

- **Code Interpreter + 웹 브라우징 조합**: Phase 2 스크리닝이 가장 빠름
- **GPTs Memory**: 매주 같은 산업 추적 시 직전 Top Pick 결과 자동 인용
- **DALL-E**: 발표 임팩트용 일러스트 (단, 학회 양식 확인)

---

## Phase별 ChatGPT 활용 팁

### Phase 1 — 산업 진단 (웹 브라우징 풀가동)
```
"최근 30일 [산업명] 산업 리포트 5건 자동 검색 (한경 컨센서스·증권사 사이트).
각 리포트의 시장 규모·CAGR·핵심 트렌드 자동 추출.
출처별 수치 차이가 있으면 표로 비교 + [검증 필요] 태그"
```

### Phase 2 — 스크리닝 (Code Interpreter)
```
"후보 종목 [기업1, 기업2, ...]의 시총·매출·OPM·ROE·PER을 웹에서 자동 수집.
Python으로 점수 계산 (가중치 30/25/20/15/10).
정렬해 Top 3 자동 추출. Excel 파일로 다운로드 링크 제공"
```

### Phase 4 — Top Pick 정식 9단계 압축
ChatGPT는 한 번에 너무 많이 요청하면 누락 발생 → Phase 4를 A·B·C·D·E 5개 메시지로 분할 권장.

## 자주 묻는 질문

**Q: 후보 종목이 10개를 넘으면?**
A: Phase 2 스크리닝을 2단계로:
  1차: 10~20개에서 점수 상위 5~7개로 압축
  2차: 5~7개에서 정밀 점수 + Red Team 통과 가능성 검토 → Top 1~2

**Q: ChatGPT 무료에서 Top Pick 가능?**
A: Phase 1·3·6은 가능. Phase 2 Code Interpreter·웹 브라우징은 Plus 권장.
