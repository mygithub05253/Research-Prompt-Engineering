## ChatGPT 위클리 프롬프트 (전체 복사 → ChatGPT)

> 인풋: `위클리_투자리포트_클로드.md`의 프롬프트와 본문 동일
> ChatGPT 차이점만 아래에 추가

```
[Step 0 ChatGPT 추가 지침]

1. **웹 브라우징 활성** (Step 0 요청)
   - 이번 주 새 정보 5개 카테고리(공시·실적·뉴스·거시·컨센서스) 모두 자동 검색
   - 각 항목에 출처 URL + 검색 결과 날짜 명시

2. **Code Interpreter 활용 (선택)**
   - 직전 → 이번 주 변동 표를 Python으로 자동 계산
   - 컨센서스 분포 시각화 (히스토그램)

3. **GPTs 활용 권장**
   - 학회 GPT(GIC Research Analyst v8.0)에 "위클리 모드 시작" 트리거 등록
   - 직전 분석 결과를 GPT Memory에 저장 → 매주 자동 인용
```

---

## ChatGPT 위클리 강점

- **웹 브라우징**: 이번 주 새 정보 자동 수집이 가장 강함
- **Code Interpreter**: 정량 변동 표 자동 계산
- **GPTs Memory**: 직전 분석을 자동 기억해 위클리 빠르게 갱신

## 위클리 4페이지 양식 (마스터)

위 클로드 버전과 동일. 다음 항목만 ChatGPT 적용 시 보강:

### 2페이지 — 이번 주 새 정보 정리 (ChatGPT 강화)

```
[웹 브라우징 명령]
"최근 7일간 [기업명] 관련 한국·영문 뉴스 자동 검색.
공시·실적·산업·거시·컨센서스 5개 카테고리로 분류.
각 항목 핵심 1문장 + URL + 발행일 명시.
검색 결과 0건이면 '이번 주 변동 없음' 표기"
```

### 3페이지 — 멀티플 변동 자동 계산 (ChatGPT 강화)

```
[Code Interpreter 명령]
"직전 분석의 EPS·멀티플 vs 이번 주 컨센서스 평균을 표로 비교.
변동률 자동 계산. ±5% 이상 변동 시 [주목] 태그.
직전 → 이번 주 변동 시각화 (간단 막대 그래프)"
```
