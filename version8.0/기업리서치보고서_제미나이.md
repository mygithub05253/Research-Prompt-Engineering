   - 단, 매 단계 답변이 길어지면 토큰 소진 빠름 → 단계 사이에 불필요 부분 정리

   - Step 6 5개년 모델: "이 표를 Google Sheets에 만들어줘" → 시트 자동 생성
   - Balance Check 수식 자동 삽입

5. **블록 D(Mermaid) 주의**
   - Gemini는 Mermaid 미리보기 미지원 (Claude·ChatGPT 동일)
   - 코드 블록만 출력하고 mermaid.live에서 변환

6. **블록 F(Red Team) 보강**
   - Gemini는 부정적 톤에 다소 신중함 → "회의주의 모드 활성, 공격 톤 허용" 명시
   - → `Step5_5_RedTeam_v8.md` 참조

## 단계별 Gemini 팁

### Step 1 — 피어 매핑 (Search Grounding)
> "Google Search Grounding 활성. [기업명]의 국내 + 글로벌 피어 6개사를 최신 시가총액·매출 기준으로 표로 정리. 각 피어의 출처 URL 명시"

→ Gemini가 자동 검색 + 출처 URL 첨부.

### Step 2 — 재무 분석 (대용량 PDF)
첨부: 사업보고서 PDF (100페이지+)
> "첨부 PDF의 III장 재무제표 + II장 사업의 내용에서 5개년 IS/BS/CF + 사업부별 매출 추출. 마크다운 표로 정리"

→ Gemini Pro의 2M 토큰으로 통째 처리.

### Step 3 — 산업 분석 (Search Grounding + 이미지)
> "TrendForce·Yole·KIET 등에서 [시장명] 글로벌 시장 규모 검색. 검색 결과 그래프가 있으면 이미지로 가져와 추이 정리. CAGR 산출 및 출처 URL 명시"

→ Search + Multimodal 조합.

### Step 4 — 최근 이슈 (Search Grounding 활용)
> "최근 3개월간 [기업명] 관련 한국·영문 뉴스 5건을 시간 순으로 정리. 각 기사 핵심 1문장 + URL"

→ Gemini Search 가장 잘하는 영역.

### Step 5 — 투자포인트 + Evidence Card
> "투자포인트 3개. 각 포인트의 Evidence Card 근거 3건은 웹 검색으로 1차 출처에서 가져오고, 답변 끝에 모든 출처 URL 모음 표 첨부"

→ Search Grounding이 자동으로 URL 첨부.

### Step 6 — 5개년 모델 (Google Sheets 연동)
> "5개년 IS/BS/CF를 Google Sheets로 생성. Bear/Base/Bull 시나리오 각 시트. Balance Check 수식 (=B10-B11-B12) 포함. 시트 공유 링크 제공"

→ Gemini Advanced에서만 가능. 일반 Gemini는 마크다운 표.

### Step 8 — 슬라이드별 복붙 (Search 비활성)
> "Step 8은 Search Grounding 끄고 텍스트만 출력. 글자 수 제한 자동 측정 후 보고"

→ Step 8 출력 단순화.

## v7.0 웹앱 폐기 안내 (중요)

v7.0의 index.html은 **사용하지 마세요**.
- Gemini 2.0 Flash는 2026.03.03자로 deprecated
- 무료 한도가 0이 되어 429 오류 발생
- v8.0은 Gemini Chat (gemini.google.com)에 직접 복붙

**Gemini Chat 사용법**:
1. gemini.google.com 접속 (구글 계정 로그인)
2. 마스터 시스템의 Step 0 ``` 블록 ``` 복사 → Gemini Chat에 붙여넣기
3. `[대괄호]` 부분 본인 분석 대상으로 수정 후 전송
4. 답변 검토 후 Step 1 동일하게 반복
5. 9단계 모두 한 세션에서 진행 (컨텍스트 유지)

**API 키·웹앱 모두 불필요**. 무료 Gemini로도 충분.

## 자주 묻는 질문 (Gemini 사용 부원)

**Q: Gemini 무료 버전(gemini.google.com)에서도 9단계 가능?**
A: 가능. 단 Search Grounding 횟수·이미지 분석 횟수 일일 제한. 핵심 단계(Step 3·5·6)에서는 Advanced 권장.

**Q: Google Sheets 자동 생성은 Gemini 무료에서도?**
A: 안 됨. Gemini Advanced(유료)에서만 Workspace 연동.

**Q: 답변 길이 제한이 있나?**
A: Gemini Pro는 2M 토큰까지 입력, 출력은 단일 응답 약 8K 토큰. 9단계 전체를 한 응답에 요청하면 잘림 → 반드시 단계별 분할.

**Q: Gemini의 답변이 종종 한국 회계 기준(K-IFRS)과 안 맞을 때?**
A: Step 0에서 "한국 회계 기준(K-IFRS) 별도재무제표 기준" 명시. DART 사업보고서 PDF 첨부 시 K-IFRS 자동 인식.
