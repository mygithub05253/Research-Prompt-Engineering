## Gemini 추가 지침

```
[Step 0 Gemini 추가 — Top Pick 전용]

1. **Google Search Grounding 풀가동**
   - Phase 1 산업 진단: 최신 산업 동향·정책·M&A 자동 검색
   - Phase 2 스크리닝: 후보 종목 멀티플 실시간 검색
   - Phase 4 Top Pick: 최근 3개월 뉴스 + 출처 URL 자동 첨부

2. **2M 토큰 컨텍스트 활용**
   - 후보 5~10개의 사업보고서 PDF 모두 첨부 가능
   - 한 세션에서 전체 산업 종합 분석

3. **유튜브 분석 (Phase 1·4)**
   - 산업 컨퍼런스·실적 발표 영상 URL → 자동 트랜스크립트 + 핵심 요약
   - 후보 종목 IR 영상 다중 분석

4. **Google Sheets 연동 (Advanced)**
   - Phase 2 스크리닝 점수 표를 Google Sheets에 자동 생성
   - 매주 누적해 산업별 Top Pick 트래킹 가능

5. **이미지 분석**
   - 후보 종목 사업보고서의 차트·그림 이미지 직접 분석
   - 증권사 리포트의 그래프에서 수치 추출
```

---

## Gemini Top Pick 강점

- **Search Grounding**: Phase 1·4의 출처 URL 자동 첨부 — Top Pick의 신뢰도 ↑
- **유튜브 분석**: 산업 컨퍼런스·IR 영상 → 다른 AI가 못하는 영역
- **2M 토큰**: 후보 5~10개 사업보고서 PDF 통째 인풋
- **Google Sheets 연동**: 매주 누적 트래킹 자동화 (Advanced)

## 자주 묻는 질문

**Q: Gemini 무료에서 Top Pick 가능?**
A: Phase 1·6은 가능. Phase 2 스크리닝은 Search 횟수 많아 Advanced 권장.

**Q: 사업보고서 5~10개 첨부 시 토큰 한계?**
A: Gemini Pro 2M 토큰은 사업보고서 50~100개도 수용. 단 응답 토큰은 8K 제한 → 분할 출력 권장.

**Q: 유튜브 영상이 한국어인데 분석 가능?**
A: 한국어 영상 분석 가능. 단 자동 자막이 정확하지 않을 때 → 영상 발표자료 PDF 첨부 병행 권장.
