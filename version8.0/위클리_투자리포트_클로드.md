# 위클리 투자리포트 — Claude 최적화 (v8.0)

> **출력물**: 4페이지 압축 위클리 (네이버증권 위클리 양식 참고)

> **차이점**: 매주 새 정보 위주, 직전 분석 결과를 베이스로 사용

---

## 위클리의 본질

- **타깃 독자**: 시장을 매주 따라가지만 시간 부족한 투자자
- **분량**: A4 4페이지 / 발표 4슬라이드
- **빈도**: 주 1회
- **인풋**: 직전 분석 + 이번 주 새 정보
- **아웃풋**: 핵심 변동 + 액션 (BUY 유지·HOLD 전환 등)

## Claude 위클리 강점

- **Artifacts**: 직전 → 이번 주 변동 표를 인터랙티브 비교 차트로
- **회의주의**: 컨센서스 변화에 대한 자체 판단 추가 가능
- **PDF 첨부**: 이번 주 발표된 IR 자료·분기 실적 PDF 직접 분석

## 자주 묻는 질문

**Q: 직전 분석이 없으면?**
A: 정식 9단계(`GIC_Research_Prompt_System_v8.md`) 먼저 1회 실행. 이후부터 위클리.

**Q: 이번 주에 새 정보가 거의 없을 때?**
A: "이번 주 정보 변동 없음. 직전 의견 유지" 1줄로 마무리. 4페이지 강제 채울 필요 없음.

**Q: 위클리에도 Red Team 필요?**
A: 직전 분석에서 통과한 Red Team 결과 1줄 인용으로 갈음. 단, 이번 주 새 정보가 직전 의견을 위협하면 Red Team 재실행 권장.
