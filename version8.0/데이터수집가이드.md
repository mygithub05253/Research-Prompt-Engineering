## 1. DART 전자공시시스템 — 최강 공식 출처

URL: `https://dart.fss.or.kr`

### 받을 수 있는 자료 (무료, 가입 불필요)
- **사업보고서** (연간) — 가장 풍부, 사업부별 매출까지
- **분기보고서** (1·3분기), **반기보고서** (2분기)
- **주요사항보고서** — M&A, 자사주, 유상증자
- **감사보고서** — 재무제표 원본
- **임원·주요주주보유주식변동상황보고서**

### 사용법
1. 검색창에 기업명 입력 (예: "삼성전기")
2. 좌측 [정기공시] 클릭 → 사업보고서 클릭
3. 우측 상단 PDF 다운로드 또는 본문 직접 열람
4. 본문 [II. 사업의 내용] → 사업부별 매출
5. 본문 [III. 재무에 관한 사항] → 재무제표 원본
6. 본문 [V. 회계감사인의 감사의견 등] → 감사 의견

### 검색 팁
- DART의 "기업개황" 메뉴 → 종목코드·자본금·발행주식수 한 페이지 정리
- "비교공시" 메뉴 → 동종업종 기업 재무제표 한꺼번에 비교

### v8.0 활용
- **Step 2**: 재무제표 3~5개년 (사업보고서 PDF 첨부 권장)
- **Step 4**: 사업부 매출 비중·핵심 고객·R&D
- **Step 6**: 발행주식수·자사주·순차입금
- **Quick Track**: 사업보고서 PDF 1개를 AI에 첨부

## 2. 한국거래소(KRX) 시장 정보

URL: `data.krx.co.kr`

### 받을 수 있는 자료 (무료, 가입 불필요)
- 종목별 일별 OHLCV (시가·고가·저가·종가·거래량)
- 시가총액·외국인 보유 비중·거래대금 상위
- ETF·ETN·선물·옵션 데이터
- 산업별·테마별 종목 리스트

### 사용법
1. 좌측 메뉴 [기본통계] → [주식] → [종목시세]
2. 종목 검색 후 기간 설정
3. 우측 상단 [Excel] 또는 [CSV] 다운로드

### v8.0 활용
- **Step 6 민감도 분석**: 1년 주가 변동성 (β 추정용)
- **Step 7 검토**: 직전 거래일 종가 확인
- **Step 1**: 시가총액 비교

## 3. 네이버 금융 — 가장 빠른 1차 스크리닝

URL: `finance.naver.com`

### 받을 수 있는 자료 (무료, 가입 불필요)
- 현재가·시총·52주 고저·외국인 비율
- 컨센서스 PER·PBR·EPS·BPS
- 분기·연간 재무제표 (3~10년치)
- 동종업종 기업 비교
- 종목토론실 (시장 분위기 파악)

### 사용법
1. 종목 검색 (예: "삼성전기")
2. 우측 상단 시세 박스 → 현재가·시총
3. 좌측 [종목분석] → [재무분석] → 재무제표 표
4. [종목분석] → [동일업종비교] → 피어 자동 매칭

### 한계
- 컨센서스가 FnGuide만큼 정밀하지 않음
- 분기 재무제표는 K-IFRS 별도/연결 구분 모호할 때 있음
- 사업부별 매출 없음 (DART 사업보고서로 보완)

### v8.0 활용
- **Step 0**: 시가총액·현재가·52주 고저
- **Step 1**: 동일업종비교로 피어 1차 추출
- **Step 6**: 컨센서스 PER·EPS

## 4. FnGuide Company Guide — 증권사도 보는 정밀 데이터

URL: `comp.fnguide.com`

### 받을 수 있는 자료 (무료, 가입 불필요)
- 5~10년 재무제표 (K-IFRS 별도·연결 구분)
- 컨센서스 NTM EPS·PER (가장 신뢰도 높음)
- 사업부별 매출 (DART 사업보고서 추출본)
- ROE·ROIC·잉여현금흐름 자동 계산
- 분기 실적 발표 후 컨센서스 변화 추이

### 사용법
1. 종목 검색
2. [Snapshot] 탭 → 한 페이지 종합
3. [기업개요] → 사업부 매출 비중
4. [재무제표] → 별도/연결 선택 후 5~10년 표
5. [Valuation] → PER·EV/EBITDA 추세

### v8.0 활용
- **Step 2**: 재무제표·핵심 지표 자동 산출본
- **Step 6**: 컨센서스 NTM EPS (목표주가 산식의 핵심 인풋)
- **Step 1**: 동종업종 멀티플 평균

## 6. 증권사 리서치 리포트 — 컨센서스의 원천

### 출처
- **한경 컨센서스** (`consensus.hankyung.com`) — 무료, 가입 불필요
- **네이버 증권 리서치 메뉴**
- **각 증권사 공식 사이트** (가입 무료)
  - 미래에셋·NH투자·키움·하나·삼성·KB·대신·메리츠 등

### 사용법
1. 한경 컨센서스 → 종목 검색 → 최근 30일 리포트 PDF 다운
2. 리포트 첫 페이지 우측 박스 = 목표주가·투자의견·EPS 추정치
3. 본문 = 산업·기업 분석 (Step 3·4 베이스)
4. 부록 = 5개년 추정 재무제표 (Step 6 베이스)

### v8.0 활용
- **Step 3**: 산업 분석의 1차 베이스 (3~5개사 리포트 종합)
- **Step 5**: 투자포인트 후보 풀
- **Step 6**: 컨센서스 목표주가 평균

### 주의
- 증권사 리포트는 **BUY 편향** (목표주가 대비 매도 의견 매우 드묾)
- 따라서 Step 5.5 Red Team이 더 중요
- 동일 종목에 다른 의견 있을 때 평균 + 분포 모두 명시

## 10. 데이터 수집 워크플로 (실전)

### 분석 시작 전 (5분)
1. 네이버 금융에서 [종목명] 검색 → 시가총액·현재가·외국인비율 확보
2. DART에서 가장 최근 사업보고서 PDF 다운로드
3. FnGuide Company Guide → Snapshot 탭 캡처

### Step 0~2 (15분)
1. 사업보고서 PDF를 AI에 첨부 (Claude/ChatGPT-Plus/Gemini)
2. 위 5분 캡처본을 본문에 붙여넣기
3. 마스터 시스템의 Step 0~2 프롬프트 차례로 실행

### Step 3 산업 (15분)
1. 한경 컨센서스에서 동종업종 리포트 3건 다운
2. KIET·TrendForce 등 산업 리포트 시장 규모 1건 확보
3. Step 3 프롬프트 실행 시 "참고: [출처별 수치]"로 본문에 넣기

### Step 5·6 (20분)
1. FnGuide → 컨센서스 NTM EPS 확보
2. 인베스팅닷컴 → 글로벌 피어 4개사 멀티플 확보
3. 마스터 시스템 Step 5·5.5·6 프롬프트 차례로 실행

### Step 7·8 (10분)
1. 직전 거래일 종가 재확인 (네이버금융)
2. Step 7 점검 → "확정" → Step 8 복붙 가이드 받기
3. PPT 양식에 슬라이드별로 복사

**총 1시간 5분** — 정식 9단계 + Red Team 포함

## 자주 묻는 질문

**Q: DART API를 쓰면 더 빠르지 않나?**
A: API 키 발급·Python 셋업 필요. 비전공자 부원 진입장벽 높음. v8.0은 코드 의존 0이 원칙. 단, 자동화에 익숙한 부원은 v6.0의 Python 코드 예시를 그대로 사용해도 됨.

**Q: 무료 자료만으로 증권사 PB급 분석이 가능한가?**
A: 가능. 증권사 리서치팀도 1차 자료는 DART·FnGuide·블룸버그(유료)를 쓰는데, FnGuide·인베스팅닷컴이 블룸버그 80% 수준. 차이는 실시간성과 일부 비상장 데이터.

**Q: 사업보고서 PDF가 100페이지 넘는데 AI가 다 읽나?**
A: Claude(200K 토큰)·Gemini Advanced(2M 토큰)는 거뜬. ChatGPT Plus는 약 50페이지가 안전선 — 핵심 부분만 추려서 첨부 권장. Quick Track은 [II. 사업의 내용] + [III. 재무에 관한 사항]만 분리해 첨부해도 충분.

**Q: 글로벌 피어가 사업 모델이 다르면?**
A: Step 1에서 유사도 [상/중/하] 표기. [중]·[하] 피어는 멀티플 평균 산출 시 가중치를 낮추거나 제외 명시.
