## 단계별 ChatGPT 팁

### Step 0 — 초기 설정
> "한국어로 답변. 모든 수치에 출처 명시. 웹 브라우징 활성. Code Interpreter 활성"

→ 한 번에 모드 설정.

### Step 2 — 재무 분석 (Code Interpreter 활용)
첨부: 사업보고서 PDF
> "Code Interpreter로 PDF의 III장 재무제표를 추출해 5개년 표로 만들어줘. 그 후 ROE/ROIC/EBITDA를 계산하고 마크다운 표로 출력해줘"

→ ChatGPT가 자동으로 Python 코드 실행 → 정확한 표.

### Step 3 — 산업 분석 (웹 브라우징)
> "TrendForce·Yole·KIET 3곳에서 [시장명] 시장 규모를 검색해 표로 정리. 출처 URL과 날짜 명시"

→ ChatGPT의 웹 검색 능력 활용.

### Step 5 — 투자포인트 + Evidence Card
> "투자포인트 3개. 각 포인트의 Evidence Card 근거 3건은 웹 검색으로 1차 출처(공시·기업 IR·정부 통계) 위주로 가져와줘"

→ ChatGPT의 출처 추적 능력 활용.

### Step 6 — 5개년 모델 (Code Interpreter)
> "Code Interpreter로 5개년 IS/BS/CF를 Excel 파일로 생성. Bear/Base/Bull 시나리오 각각의 시트. Balance Check 수식 포함. 다운로드 링크 제공"

→ Excel 파일 다운로드 가능 (단, Step 8 슬라이드 복붙용은 별도 텍스트).

### Step 8 — 슬라이드별 복붙 (Code Interpreter 비활성)
> "Step 8은 Code Interpreter·웹 브라우징 모두 끄고 일반 텍스트로 슬라이드별 박스만 출력. 글자 수 제한 매 박스 자동 측정"

→ 부원이 그대로 슬라이드에 복붙.

## 자주 묻는 질문 (ChatGPT 사용 부원)

**Q: ChatGPT 무료 버전(GPT-4o-mini)에서도 가능?**
A: Step 0~5는 가능. 단 Code Interpreter·고급 웹 브라우징은 Plus 권장. 무료에서는 Quick Track 권장.

**Q: Code Interpreter가 사업보고서 PDF 100페이지를 다 읽나?**
A: 텍스트 추출은 가능하나 50~80페이지가 안전선. III장 재무·II장 사업 부분만 잘라서 첨부 권장.

**Q: GPTs를 학회 외부에 공개해도 되나?**
A: 무료 GPT는 ChatGPT Plus 사용자만 사용 가능. 외부 공개 시 학회 명의·면책 문구 필수.
