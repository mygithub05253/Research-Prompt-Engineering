## Claude의 강점 (v8.0에서 활용)

| Claude 강점 | v8.0에서 어떻게 활용 |
|---|---|
| **Artifacts** (HTML·표·코드 인터랙티브 미리보기) | Step 6 시나리오 매트릭스를 Artifact로 → 부원이 슬라이더로 가정 변경 |

| **표·서술 균형** | Step 6 표 + 서술 동시 출력에 강점 |

## Claude 전용 모델 메모 (Step 0에 추가)

```
[Claude 전용 추가 지침]

1. **Artifacts 적극 활용**
   - Step 2 재무 분석: HTML Artifact로 매출·OPM 추이 차트 (Chart.js 가능)
   - Step 6 시나리오 매트릭스: 인터랙티브 표 Artifact (가정 변경 시 자동 재계산)
   - Step 8 슬라이드별 텍스트: 일반 텍스트 (Artifact 사용 X — 복붙해야 함)

2. **사업보고서 PDF 첨부 시**
   - Claude는 PDF 직접 파싱 가능
   - 첨부 직후 "재무제표 III장과 사업의 내용 II장 위주로 추출"이라고 명시

3. **블록 G(Evidence Card) 강화**
   - Claude의 "근거 비교" 능력을 활용해 신뢰도 [상/중/하] 판정의 근거를 추가 1줄로 명시
   - 예) "신뢰도 상 — 출처 3건 모두 1차 출처(IR·DART·정부 통계)"

4. **블록 F(Red Team) 강화**
   - Claude는 회의주의 페르소나에 자연스럽게 들어감
   - 페르소나 카드 1(Wall Street 베테랑)을 기본 + 카드 4(Quant)을 보조로 추가 권장
   - → `Step5_5_RedTeam_v8.md` 참조

5. **결론 → 근거 구조 유지**
   - Claude의 답변 패턴은 자연스럽게 "결론 + 근거" 순서
   - Step 7 검토에서 이 순서로 출력
```

## 단계별 Claude 팁

### Step 2 — 재무 분석
> "Artifact로 매출·영업이익·OPM 추이 차트를 만들어줘. 그 후 같은 데이터의 마크다운 표를 따로 출력해줘"

→ 발표용 캡처본(Artifact 차트) + Step 8 복붙용 표(마크다운) 둘 다 확보.

### Step 3 — 산업 분석
> "글로벌 [시장명] 시장 규모를 TrendForce·Yole·KIET 3곳에서 교차 인용해줘. 출처별 수치 차이가 있으면 표로 보여주고 [검증 필요] 태그를 붙여줘"

→ Claude의 다출처 교차 검증 능력 활용.

### Step 5 — 투자포인트 + Evidence Card
> "투자포인트 3개 도출. 각 포인트에 Evidence Card(블록 G) 의무 적용. 신뢰도 [상/중/하] 판정 시 그 판정의 근거를 1줄 추가해줘"

→ Claude는 "왜 상이라고 판정했나"를 명시적으로 답변.

### Step 5.5 — Red Team
> "두 페르소나로 Red Team. 1) Wall Street 베테랑 Short Seller (논리·데이터 공격) 2) Quant 분석가 (회계·이상치 공격). 둘이 합쳐 3개 공격을 만들어줘"

→ Claude의 자연스러운 페르소나 전환 활용.

### Step 6 — 시나리오 매트릭스
> "시나리오 매트릭스를 Artifact로 만들어줘. 가정 슬라이더 (매출 성장률 ±5%, OPM ±2%)를 두면 EPS·목표주가가 자동 재계산되도록"

→ Claude Artifact의 React/JS 능력 활용.

### Step 8 — 슬라이드별 복붙
> "주의: Step 8은 Artifact 사용 금지. 일반 텍스트로 슬라이드별 박스만 출력. 글자 수 제한(≤NN자)을 매 박스마다 자동 측정해 보고해줘"

→ 부원이 슬라이드에 그대로 복붙 가능한 형태.

## 자주 묻는 질문 (Claude 사용 부원)

**Q: Claude Artifacts는 슬라이드에 그대로 들어가나?**
A: 아니. Artifact는 분석·검토용이고, 발표 슬라이드에는 차트 캡처본 + 마크다운 표 텍스트가 들어감.

**Q: Claude 무료 버전(claude.ai 무료)에서도 9단계 가능?**
A: 가능. 단 Artifacts 일부 기능 제한. 사업보고서 PDF 첨부는 무료 버전에서도 됨.

**Q: Claude의 "사고 과정 표시(Extended Thinking)"는 켜는 게 좋나?**
A: Step 5(투자포인트 도출)·Step 5.5(Red Team)·Step 6(밸류에이션 가정)에서 켜기 추천. Step 0~2·8은 끄는 게 빠름.
