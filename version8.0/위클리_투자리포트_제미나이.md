## Gemini 위클리 프롬프트 (전체 복사 → Gemini)

> 본문은 `위클리_투자리포트_클로드.md`와 동일
> Gemini 차이점만 아래에 추가

```
[Step 0 Gemini 추가 지침]

1. **Google Search Grounding 풀가동**
   - 이번 주 새 정보 5개 카테고리(공시·실적·뉴스·거시·컨센서스) 자동 검색
   - 답변 끝에 출처 URL 자동 첨부 (Gemini 강점)
   - "최근 7일 한정 검색"으로 기간 명시

2. **유튜브 분석 (선택)**
   - 이번 주 발표된 실적·IR Day 영상 URL → "이 영상 핵심 5분 요약"
   - Gemini가 자동 트랜스크립트 추출 + 요약

3. **Google Sheets 연동 (Advanced)**
   - 직전 → 이번 주 변동 표를 Google Sheets에 자동 기록
   - 매주 누적 → 분기 후 추세 분석에 활용

4. **2M 토큰 컨텍스트**
   - 직전 분석 PDF + 이번 주 발표 IR PDF + 분기 실적 PDF 모두 첨부 가능
   - 한 세션에서 종합 위클리 생성
```

---

## Gemini 위클리 강점

- **Search Grounding**: 자동 출처 URL 첨부 — 위클리 신속성에 최적
- **유튜브 분석**: 실적 발표 영상 직접 분석 (다른 AI 미지원)
- **2M 토큰**: 누적 위클리·분기 실적·산업 리포트 통째 인풋

## 위클리 4페이지 양식 (Gemini 보강 부분)

### 2페이지 — 이번 주 새 정보 (Search 활용)

```
[Search Grounding 명령]
"최근 7일간 site:dart.fss.or.kr OR site:hankyung.com [기업명] 검색.
공시·언론 5건. 각 항목 1문장 요약 + 발행일 + URL.
검색 결과 0건이면 '이번 주 신규 정보 없음' 표기"
```

### 3페이지 — 컨센서스 변화 (실시간 검색)

```
[Search 명령]
"한경 컨센서스에서 [기업명] 최근 30일 신규 리포트 자동 검색.
직전 위클리 시점 대비 컨센서스 EPS·목표주가 평균 변동 계산.
± 5% 이상 변동 시 [주목] 태그"
```

## 자주 묻는 질문

**Q: Gemini 무료에서 위클리 가능?**
A: 가능. Search Grounding 일일 횟수 제한이 위클리(주 1회)에는 충분.

**Q: 누적 위클리 자동 정리?**
A: Gemini Advanced + Google Drive 연동 시 매주 출력본 자동 저장. 분기 결산 시 "최근 12주 위클리 요약"으로 추세 분석 가능.
